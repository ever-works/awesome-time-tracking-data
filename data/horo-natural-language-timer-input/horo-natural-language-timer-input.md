## The Speed Advantage

Horo's natural language input eliminates the friction of traditional timer setup, allowing users to create timers in under a second.

## Supported Input Formats

### Duration-Based
- **"1:30:45"** → 1 hour, 30 minutes, 45 seconds
- **"1.5h"** → 1 hour, 30 minutes
- **"45m"** → 45-minute timer
- **"90s"** → 90 seconds
- **"2h30m"** → 2 hours, 30 minutes

### Time-Based
- **"@3pm"** → Countdown to 3:00 PM
- **"@15:30"** → Countdown to 3:30 PM (24-hour format)
- **"@5"** → Countdown to 5:00 (AM or PM, whichever comes next)

### Natural Phrases
- **"pomodoro"** → 25-minute timer
- **"break"** → 5-minute timer
- **"lunch"** → 1-hour timer

## Why Natural Language Matters

### Reduced Cognitive Load
- No mental translation needed
- Think in natural time units
- Type exactly what you're thinking
- Immediate understanding

### Speed of Execution
- Type and press Enter - done
- No clicking through menus
- No selecting hours/minutes separately
- No cursor positioning

### Lower Barrier to Use
- Easy to remember format
- Forgiving of variations
- Intuitive for first-time users
- No manual to read

## Use Cases

### Pomodoro Technique
Quickly set 25m work sessions and 5m breaks without breaking flow

### Meeting Time Tracking
Type "@10:30" when meeting starts at 9:45 to track remaining time

### Cooking/Breaks
Rapid timer creation for any duration
