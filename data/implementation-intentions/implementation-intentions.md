## Overview

Implementation Intentions pre-decide actions using if-then format, automating behavior execution.

## Format

"If [situation], then I will [action]"

## Examples

- If it's 9am Monday, then I'll work on strategic planning
- If I finish lunch, then I'll review today's priorities
- If meeting ends early, then I'll do 10min email

## Research

Peter Gollwitzer's studies show 2-3x higher goal completion rates with implementation intentions vs vague goals.

## Benefits

- Removes decision moment
- Creates automatic triggers
- Increases follow-through
- Reduces procrastination
