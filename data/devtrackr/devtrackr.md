## Overview

DevTrackr is an automatic time tracking solution built specifically for software developers. The platform tracks coding time automatically across VS Code, JetBrains IDEs (IntelliJ, PyCharm, WebStorm), and Visual Studio.

## Features

- **Multi-IDE Support**: Works seamlessly across popular development environments including VS Code, JetBrains suite, and Visual Studio
- **Automatic Tracking**: Passively monitors coding activity without manual timer management
- **Project Analytics**: Provides insights into time spent per project, language, and file
- **Privacy-Focused**: Designed with developer privacy in mind
- **Detailed Reports**: Generate comprehensive reports on coding patterns and productivity

## Use Cases

- Freelance developers tracking billable hours across multiple projects
- Development teams measuring project time allocation
- Individual developers analyzing their coding patterns
- Agencies billing clients based on actual development time
