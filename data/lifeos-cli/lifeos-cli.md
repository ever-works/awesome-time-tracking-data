## Overview

lifeos-cli is a terminal-native LifeOS application for managing intentions, habits, and timelogs.

## Features

- Terminal-native interface
- Manage intentions, habits, and time logs
- Agent-friendly interface
- PostgreSQL backend for data storage
- Open source
- Command line interface

## Use Cases

- Personal productivity tracking
- Habit formation tracking
- Intentions setting and review
- Time logging for projects or activities
