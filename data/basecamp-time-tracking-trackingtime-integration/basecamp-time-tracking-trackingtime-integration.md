# Basecamp Time Tracking (TrackingTime Integration)

## Description
TrackingTime’s Basecamp 4 integration lets teams track time directly on Basecamp projects and tasks via a browser extension. It consolidates project work and time data, generates automatic timesheets, and provides analytics through the TrackingTime platform.

## Features
- **Direct time tracking in Basecamp 4**: Start and stop timers from within your Basecamp 4 workspace.
- **Browser extension support**: Time tracking via the TrackingTime Button extension for:
  - Chrome
  - Firefox
  - Safari
  - Microsoft Edge
- **Automatic timesheets**: Logged time is automatically consolidated into timesheets in TrackingTime.
- **In-depth analytics**: Access detailed time and project analytics through TrackingTime’s reporting tools.
- **Track time on projects and tasks**: Associate tracked time with specific Basecamp 4 projects and related work items.
- **All-in-one operations tracking (via TrackingTime)**:
  - Time tracking across projects and clients
  - Project monitoring and progress oversight
  - Support for complying with internal and external time-reporting requirements
  - Flexible reporting on activities, projects, and workloads
- **Multi-app workflow support**: Use the same extension to track time in other supported productivity tools, reducing the need to switch between apps.
- **Fast setup**: Connect TrackingTime to Basecamp 4 and start tracking within a few minutes (sign up, install extension, log into Basecamp 4).

## Integrations
- **Basecamp 4**: Native-style time tracking added via the browser extension.
- **Other apps (via TrackingTime)**: The TrackingTime Button also works in various other productivity tools (not listed individually in the provided content), allowing unified time tracking across multiple apps.
