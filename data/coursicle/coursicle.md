## Overview

Coursicle is an app that helps students track assignments, plan class schedules, and read professor reviews. With over 2 million students using it, Coursicle provides a comprehensive solution for academic organization and planning.

## Key Features

### Assignment Tracking
- Automatically syncs and color codes assignments from multiple learning management systems
- Supported platforms: Canvas, Blackboard, Brightspace D2L, Moodle, Sakai, and Google Classroom
- Unified view of all assignments across different courses
- Reminders for upcoming classes, assignments, quizzes, projects, and exams

### Course Planning
- Build and visualize class schedules
- Create backup schedules for class registration
- Browse course catalog with filters
- View course descriptions and requirements

### Course Availability Notifications
- Get notified when seats become available in closed or waitlisted classes
- Track multiple classes simultaneously
- Does not automate registration, only notifies about availability

### Professor Reviews
- Read reviews and ratings for instructors
- Make informed decisions about course selection

## Supported Institutions

Over 1,100 colleges and universities in the United States, including:
- Community colleges
- Four-year colleges
- Public and private schools

## Platform Availability

- iOS app (App Store)
- Android app (Google Play)
- Web application (no download required)
