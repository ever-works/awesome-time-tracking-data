## Overview

Buddy Punch uses GPS tracking and geofencing to verify employee locations, ensuring workers can only clock in when physically present at designated work sites, preventing offsite punching and improving accountability for mobile workforces.

## GPS Tracking

**What It Captures**:
- Exact coordinates when employees punch in/out
- Location history throughout shift
- Route tracking for field workers
- Timestamp with location verification

**Privacy**:
- Only tracks during work hours
- Employees aware of tracking
- Location data for verification purposes
- Complies with privacy regulations

## Geofencing

**How It Works**:
1. Admin draws virtual boundaries around work sites
2. Employee attempts to clock in/out
3. System checks if GPS coordinates within geofence
4. Punch allowed if inside boundary, rejected if outside
5. Alerts manager to unauthorized punch attempts

**Flexibility**:
- Multiple geofences per company
- Different radii for different locations
- Adjust boundaries as needed
- Temporary geofences for project sites

## Prevents Common Issues

**Offsite Punching**: Cannot clock in from home/other locations

**Early/Late Arrival**: Know exactly when employee reached site

**Unauthorized Breaks**: Tracking shows if employee leaves site

**Job Site Verification**: Prove employee was at correct location
