## Overview

Productivity Momentum Building is the practice of deliberately sequencing tasks to create psychological momentum, starting with achievable wins to generate energy and confidence for larger challenges.

## The Momentum Principle

### How It Works
- Complete small, achievable task
- Experience sense of accomplishment
- Gain energy and confidence
- Apply to larger task
- Repeat and build momentum

### Scientific Basis
- Dopamine release from task completion
- Progress principle (Amabile)
- Self-efficacy building
- Reduced activation energy for subsequent tasks

## Implementation

### Morning Sequence
1. 5-minute win (make bed, quick email)
2. 15-minute task (clear inbox, review plan)
3. 30-minute focused work
4. Tackle major project with built momentum

### Task Ordering
- Start: Quick, achievable tasks
- Middle: Major deep work
- End: Medium-difficulty items

## Benefits

- Overcome procrastination
- Build daily work rhythm
- Increase completion rates
- Sustain energy longer
- Reduce decision fatigue

## Common Applications

### Writing
- Edit existing work before new writing
- Outline before detailed writing
- Warm up with easy sections

### Development
- Fix small bugs before new features
- Write tests before complex code
- Refactor familiar code first

### Creative Work
- Review previous work before creating
- Make small adjustments first
- Build on existing foundation

## Balance with Other Methods

### Eat That Frog (Opposite Approach)
- Do hardest task first
- Best for high morning energy
- Different psychological profile

### When to Use Momentum
- Struggling with motivation
- Need to overcome inertia
- First day back from break
- Low energy mornings

### When to Eat Frog
- High morning energy
- Difficult task needs peak focus
- Risk of never reaching hard task
- Single most important item
