## Overview

RescueTime automatically tracks time spent in various activities and combines productivity tools with project tracking capabilities.

## Features

- **Automatic Tracking**: Runs quietly in the background with no manual input required
- **Cross-Device Tracking**: Tracks activity across all devices
- **Website & App Tracking**: Monitors time spent in applications and websites
- **Goal Setting**: Set productivity goals and track progress
- **Distraction Blocking**: FocusTime feature to block distracting websites
- **Alerts**: Notifications when spending too much time on unproductive activities
- **Timesheets**: Automated project tracking and client report generation
- **Productivity Reports**: Detailed analytics on daily productivity patterns
