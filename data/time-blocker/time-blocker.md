## Overview

Time Blocker is a Firefox extension designed for working using the time blocking method, providing preset time blocks for work sessions and breaks.

## Time Block Presets

The extension includes three built-in presets:

1. **Work/Study Block**: Time block for focused working or studying sessions
2. **Small Break**: Short break intervals for quick rest
3. **Big Break**: Longer break periods for more substantial rest

## How It Works

Users can quickly activate different time blocks directly from the browser, helping structure work sessions according to time blocking principles. The preset system makes it easy to alternate between focus periods and breaks without manual timer configuration.

## Use Cases

- Pomodoro technique implementation in the browser
- Study session management
- Work-break cycle structuring
- Deep work session timing
- Browser-based productivity tracking

## Benefits

- Pre-configured time blocks eliminate setup time
- Browser-level integration for web-based work
- Simple interface for quick session management
- Supports healthy work-rest cycles

## Platform

Available exclusively for Mozilla Firefox as a browser extension.
