# Harvest Web App

The Harvest web app is designed to display your data in a mobile friendly format. Each web app is individually setup and customised by Harvest staff to show the things you want to see in the order you want.

## Features
- Fully customizable data display tailored to user preferences
- Mobile-friendly interface accessible via browser
- Located at app.harvest.com (not available on app stores)
- Live demo available with username 'demo' and password 'demo'
- Custom configuration by Harvest staff for individual sites

## Pricing

Pricing for the Harvest Web App is typically included in Harvest subscription plans; specific details are available from Harvest sales.

## Access

Access your custom web app at app.harvest.com after setup. For the live demo, use username 'demo' and password 'demo'.

## Getting Started

Contact Harvest to request a custom web app for your own Harvest site.
