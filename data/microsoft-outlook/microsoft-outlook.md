## Overview

Microsoft Outlook is an email client and personal information manager from Microsoft. It combines email, calendar, task management, and note-taking capabilities into a single application, serving as a central hub for workplace communication and organization.

## Features

- Email management with advanced organization features
- Integrated calendar for scheduling and meeting management
- Task and to-do list management
- Note-taking capabilities
- Contact management
- Multiple account support
- Integration with Microsoft 365 ecosystem

## Pricing

Free version available. Microsoft 365 subscriptions starting at $6.99/month for personal plans include expanded features and storage.
