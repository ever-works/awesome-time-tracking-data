## Overview

Pommie is a feature-rich Pomodoro timer application for macOS that combines simplicity with powerful features. Available on the Mac App Store, it offers extensive customization options while maintaining an elegant, native Mac interface.

## Features

### Menu Bar Integration
- Show or hide countdown timer in menu bar
- Quick access from anywhere
- Visual countdown display
- Unobtrusive design

### Keyboard Shortcuts
- Configurable system-wide keyboard shortcuts
- Quick start/stop timer control
- Open menu from anywhere
- Fully customizable hotkeys

### Pomodoro Features
- Customizable work and break intervals
- Long break intervals after set number of pomodoros
- Visual and audio notifications
- Pause and resume capability
- Session history tracking

### Customization
- Adjustable timer durations
- Custom notification sounds
- Visual appearance options
- Flexible break scheduling

## Platform

- macOS only
- Available on Mac App Store
- Native Mac application
- Supports latest macOS versions

## Benefits

- **Mac Native**: Built specifically for macOS with native UI
- **Keyboard Control**: Full keyboard shortcut support for efficiency
