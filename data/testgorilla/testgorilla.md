## Overview

TestGorilla is a pre-employment testing platform that allows companies to create timed skills assessments for job candidates. It tracks time spent on tests and provides insights into candidate performance under time constraints.

## Features

- Library of 300+ pre-built tests
- Custom test creation
- Timed assessments
- Coding challenges with time limits
- Anti-cheating measures
- Candidate time tracking
- Automated scoring and ranking
- Video question responses
- Mobile-friendly tests
- Detailed performance analytics
- Integrations with ATS systems

## Pricing

TestGorilla offers tiered pricing:
- Free: Limited test library
- Starter: $75/month
- Pro: $115/month
- Custom: Enterprise pricing

Pricing based on number of candidates tested annually.
