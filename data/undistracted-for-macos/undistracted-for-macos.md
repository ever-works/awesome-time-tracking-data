## Overview

UnDistracted is a macOS application that helps you concentrate and overcome procrastination by highlighting the current working window while hiding all distracting elements on your screen with a single click.

## Key Features

### Window Highlighting
- Shadows all windows except the active one
- Fine-tune shadow levels to your preference
- Makes the current task stand out visually

### Desktop Decluttering
- Instantly hides all files, folders, and app icons on your messy desktop
- Hides Dock and menu bar
- Removes visual distractions

### Focus Mode
- Sets "Do not Disturb" mode automatically
- Blocks all distractions with one click
- Unlike other apps that perform only one task, UnDistracted removes all distracting factors simultaneously

### Advanced Automation
- Launch and close applications
- Turn system functions on/off
- Change system parameters
- Modify everything available on macOS with AppleScript, bash, and system settings

## What Makes It Unique

UnDistracted combines multiple focus features into one app - window highlighting, desktop hiding, and system-level distraction blocking - all activated with a single click.
