## Overview

Be Focused is a straightforward Pomodoro timer designed specifically for Apple devices, helping users break down work into focused intervals.

## Features

- **Pomodoro Timer**: Classic 25-minute work sessions
- **Task List**: Organize tasks alongside timer
- **Customizable Intervals**: Adjust work and break durations
- **Statistics**: Track daily and weekly productivity
- **iCloud Sync**: Synchronize between iPhone, iPad, and Mac
- **Notifications**: Alerts for session transitions
- **Goal Setting**: Set daily work targets
- **Simple Interface**: Clean, Apple-style design

## Pricing

Free version available with Pro version for advanced features.
