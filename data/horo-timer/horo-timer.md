## Overview

Horo is a timer app for the macOS menu bar that displays your timer counting down directly in the menu bar.

## Key Features

- **Menu Bar Display**: See your countdown timer progress without opening the app
- **Hashtag Organization**: Support for hashtags to sort timers into projects
- **Quick Access**: Start, stop, and manage timers from the menu bar
- **Project Tracking**: Organize time tracking by projects using hashtag system

## Use Cases

- Time-based billing for freelancers
- Project time allocation tracking
- Pomodoro technique implementation
- Meeting time management

## Platform

Available exclusively for macOS via the Mac App Store.
