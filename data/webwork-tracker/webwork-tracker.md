## Overview

WebWork Tracker is a comprehensive time tracking solution available as both a Chrome extension and desktop application. It provides automatic tracking with optional screenshot capture and activity monitoring, designed for remote teams, agencies, and freelancers who need detailed work verification.

## Features

- Chrome extension for browser-based tracking
- Desktop app for full computer monitoring
- Automatic time tracking
- Optional screenshot capture
- Activity level monitoring
- Application and website tracking
- Project and task time allocation
- Idle time detection
- Manual time entry option
- Detailed reporting and analytics

## Chrome Extension

- Track time directly from browser
- One-click start/stop
- Integration with web applications
- Lightweight and unobtrusive
- Real-time synchronization
- Works with project management tools
- Browser activity tracking

## Desktop Application

- Full computer monitoring capabilities
- Application usage tracking
- Keyboard and mouse activity
- Screenshot functionality
- Offline tracking with sync
