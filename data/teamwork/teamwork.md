## Overview

Teamwork is a project management platform with integrated time tracking capabilities designed for team collaboration.

## Features

- **Downloadable Timer**: Desktop app for Mac and Windows
- **Activity Tracking**: Monitors computer activity automatically
- **Idle Detection**: Timer stops when you're inactive
- **Project Management**: Complete project planning and task management
- **Team Collaboration**: Communication tools and file sharing
- **Task Assignment**: Distribute work across team members
- **Gantt Charts**: Visual project timeline planning
- **Resource Management**: Allocate team capacity
- **Client Access**: Portal for client collaboration
- **Reporting**: Project and time tracking reports

## Pricing

Free plan available with paid tiers for advanced features.
