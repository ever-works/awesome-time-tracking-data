## Overview

Calendar sync time tracking connects calendar applications with time tracking systems, automatically generating time entries based on calendar events. This eliminates manual entry for meetings, appointments, and scheduled work blocks.

## Key Features

- **Automatic Time Entry Creation**: Calendar events become time entries automatically
- **Bidirectional Sync**: Changes in either system reflect in both
- **Project Assignment**: Map calendar events to specific projects or clients
- **Selective Sync**: Choose which calendars or event types to track
- **Smart Detection**: Identifies work vs. personal events

## Benefits

- Eliminates manual entry for meetings and appointments
- Ensures all scheduled time is captured
- Provides accurate picture of time allocation
- Reduces timesheet completion time
- Maintains single source of truth

## Common Use Cases

- Client meetings and consultations
- Internal team meetings
- Scheduled project work blocks
- Training sessions and workshops
- Travel time between appointments
