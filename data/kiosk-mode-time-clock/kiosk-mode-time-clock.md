## Overview

Kiosk mode time clocks transform tablets, computers, or dedicated devices into shared time tracking stations where multiple employees can clock in and out. Common in retail, manufacturing, healthcare, and hospitality industries.

## Authentication Methods

### PIN Entry
- Employees enter unique personal identification numbers
- 4-6 digit codes for quick access
- Password-protected admin functions

### Facial Recognition
- Camera-based biometric identification
- Speed mode for fast clock-ins
- Prevents buddy punching
- No physical contact required

### Badge/Card Scanning
- RFID or barcode badges
- Proximity card readers
- QR code scanning
- NFC tags

### Fingerprint Scanning
- Biometric fingerprint readers
- Eliminates buddy punching
- Hardware fingerprint scanners

## Key Features

- **Single Device, Multiple Users**: One device serves entire workforce
- **Lock-Down Mode**: Restricts device to time clock function only
- **Offline Capability**: Continues functioning without internet
- **Photo Capture**: Optional photo at each clock-in
- **Department Selection**: Choose department or job code
- **Break Tracking**: Separate clock-in/out for breaks

## Benefits

- Cost-effective for multiple employees
- Centralized time tracking location
- No personal devices required
- Familiar punch clock replacement
- Easy for non-tech-savvy workers

## Hardware Options

- iPad or Android tablets with stands
- Wall-mounted touchscreen displays
- Desktop computers
- Dedicated time clock terminals
- Ruggedized devices for harsh environments
