# TrackingTime Integrations

## Overview

TrackingTime Integrations is a central directory of 60+ integrations that embed TrackingTimeâ€™s time tracking into popular web tools. It allows users to track time directly from their everyday apps in the browser, reducing the need to switch between applications while working.

Source: <https://trackingtime.co/integrations>

## Features

- **60+ native integrations** with commonly used tools across project management, development, HR, and collaboration.
- **In-app time tracking**: Start and stop timers directly from integrated tools while working in the browser.
- **Task- and project-level tracking**: Log time against specific tasks, tickets, issues, or projects in the connected app (e.g., Asana tasks, Azure DevOps work items, Basecamp projects).
- **Cross-tool consistency**: Use the same time tracking system across different platforms your team uses (e.g., PM tools, HR systems, code repositories).
- **Support for development workflows**: Time tracking on code-related work in tools like Bitbucket, Azure DevOps, Assembla, Axosoft.
- **Support for project and task management workflows**: Track time on tasks and projects in tools such as Asana, Basecamp (2, 3, 4), ActiveCollab, Agiled, Azendoo, ClickUp (partial info), Any.do, Airtable.
- **Support for HR and people operations**: Track employee time within HR platforms such as BambooHR.
- **Browser-based usage**: Designed for use while working in the browser, keeping time tracking close to where work is done.
