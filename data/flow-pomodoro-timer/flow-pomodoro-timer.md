## Overview

Flow is a minimalist Pomodoro timer application designed exclusively for macOS. It combines the proven Pomodoro Technique with optional website and app blocking to help users maintain focus during work sessions. The app prioritizes simplicity and unobtrusiveness while providing essential focus management features.

## Key Features

### Pomodoro Timer
- **Customizable Intervals**: Adjust work session and break lengths
- **Menu Bar Integration**: Always accessible from macOS menu bar
- **Floating Window**: Optional on-top window for timer visibility
- **Sound Notifications**: Audio alerts for session transitions
- **Auto-Start**: Automatically begin next session or break

### Distraction Blocking
- **Website Blocking**: Block distracting websites during focus time
- **Application Blocking**: Prevent access to non-work apps
- **Custom Block Lists**: Define which sites and apps to restrict
- **Scheduling**: Different blocks for different times
- **Break Allowance**: Automatically unblock during breaks

### Session Tracking
- **Completed Session Counter**: Track Pomodoros completed daily
- **Session History**: Review past focus sessions
- **Productivity Streaks**: Maintain consecutive days of focus work
- **Statistics**: Basic analytics on focus time

### macOS Integration

- **Native Design**: Follows macOS design guidelines
- **Menu Bar App**: Minimal screen real estate usage
- **Keyboard Shortcuts**: Quick start/stop without mouse
- **Notifications**: System-level alerts for breaks and work time
- **Dark Mode**: Respects system appearance preferences

## Workflow

1. **Start Timer**: Click menu bar icon or use keyboard shortcut
2. **Focus Work**: 25-minute default work session (customizable)
3. **Automatic Blocking**: Distracting sites/apps blocked during session
4. **Short Break**: 5-minute break after each session
5. **Long Break**: 15-minute break after 4 completed sessions
6. **Repeat**: Continue cycle throughout workday

## Customization Options

- **Work Duration**: 5-60 minutes (default 25)
- **Short Break**: 1-30 minutes (default 5)
- **Long Break**: 5-60 minutes (default 15)
- **Long Break Interval**: After 2-8 sessions (default 4)
- **Block Lists**: Custom websites and apps per focus type
- **Sounds**: Choose alert sounds or silent mode

## Comparison to Alternatives

### vs Browser Extensions
- Flow blocks at system level, not just browser
- Can't be easily disabled mid-session
- Works across all applications

### vs Tomighty (Free Alternative)
- Flow includes website/app blocking
- More polished UI/UX
- Better macOS integration
- Tomighty is completely free

### vs Forest (Gamified Option)
- Flow is more minimalist
- No gamification or tree growing
- Focus on simplicity over engagement
- Mac-only vs Forest's cross-platform

## Limitations

- **Mac Only**: No Windows or Linux support

- **Basic Analytics**: Limited compared to RescueTime or Timing
