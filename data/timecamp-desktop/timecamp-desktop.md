## Overview

TimeCamp Desktop is the desktop application component of TimeCamp, providing automatic time tracking by monitoring your computer activity. It runs in the background tracking which applications and websites you use.

## Features

- **Automatic Tracking**: Tracks apps and websites automatically
- **Activity Detection**: Monitors keyboard and mouse activity
- **Idle Time Detection**: Automatically pauses when idle
- **Manual Override**: Edit and adjust tracked time
- **Project Assignment**: Assign time to projects automatically
- **Offline Mode**: Continues tracking without internet
- **Privacy Controls**: Choose what to track
- **Sync**: Syncs with TimeCamp web and mobile
- **Lightweight**: Minimal system resource usage

## Platforms

- Windows
- macOS
- Linux
