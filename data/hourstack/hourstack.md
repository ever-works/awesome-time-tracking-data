## Overview

Hourstack is a visual time management tool that combines time blocking (planning) with time tracking (execution) in a unified calendar interface. The drag-and-drop design makes it easy to plan your day visually and then track how reality compares to your plans.

## Features

- **Visual Time Blocking**: Drag tasks onto calendar to plan your day
- **Integrated Tracking**: Track actual time spent on planned tasks
- **Planned vs Actual**: Compare estimates to reality
- **Calendar Sync**: Integrate with Google Calendar, Outlook
- **Project Management**: Organize tasks by projects and clients
- **Recurring Tasks**: Set up repeating time blocks
- **Team Scheduling**: See team capacity and availability
- **Time Reports**: Detailed reporting on time usage
- **Budgets**: Set project time budgets and track progress
- **Tags**: Categorize tasks with flexible tagging
- **Mobile App**: iOS and Android for tracking on the go
- **Integrations**: Connect with project management tools
- **Color Coding**: Visual project and category distinction
- **Drag-and-Drop**: Easy rescheduling by dragging blocks

## Workflow

1. **Plan**: Drag tasks from backlog onto calendar
2. **Estimate**: Set expected duration for each block
3. **Execute**: Track actual time using timers
4. **Compare**: See where estimates matched reality
5. **Learn**: Improve future estimates based on data

## Benefits

- Combines planning and tracking in one tool
- Visual interface reduces friction
- Learn to estimate time more accurately
- Understand where time actually goes
- Balance workload across team
