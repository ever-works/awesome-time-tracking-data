## Overview

Conjure is a sophisticated habit and time tracking platform designed for power users who want complete control over their tracking data with advanced features like custom rules, API access, and complex data relationships.

## Features

- **Flexible Tracking**: Track habits, goals, time, and custom metrics with unlimited flexibility
- **Rules Engine**: Create conditional rules and automations based on your tracking data
- **Data Layer**: Direct access to your data with SQL-like querying capabilities
- **API Access**: Full REST API for integrations and custom applications
- **Custom Dashboards**: Build personalized dashboards with widgets and visualizations
- **Cross-Platform**: Available on web, desktop (Windows, macOS, Linux), iOS, and Android
- **Data Export**: Export all data in standard formats for analysis
- **Templates**: Use pre-built templates or create custom tracking templates
- **Advanced Analytics**: Deep insights with trend analysis and correlations
- **Privacy-First**: Self-hosted option available for complete data control

## Pricing

Free tier available with basic features. Premium plans start at $8 per month with advanced rules, unlimited dashboards, and API access. Self-hosted option available for enterprise users.
