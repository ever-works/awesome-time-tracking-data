## Overview

Geckoboard is a real-time KPI dashboard platform that consolidates business metrics from multiple tools and displays them on easy-to-understand dashboards. Designed for teams that want to stay focused on their most important metrics, Geckoboard is commonly displayed on TV screens in offices or shared digitally with remote teams.

## Features

- **Real-Time Data**: Live updates from connected data sources
- **TV Dashboard Display**: Optimized for large screen displays in offices
- **80+ Integrations**: Connect to popular business tools and data sources
- **Pre-Built Widgets**: Drag-and-drop widgets for common metrics
- **Custom Visualizations**: Charts, graphs, numbers, and more
- **Team Sharing**: Share dashboards with entire teams
- **Mobile Access**: View dashboards on smartphones and tablets
- **Automated Reports**: Schedule dashboard snapshots via email
- **Multiple Dashboards**: Create different dashboards for different teams
- **Data Security**: SOC 2 Type II certified
- **No Coding Required**: Build dashboards without technical skills
- **Status Boards**: Show system status and uptime metrics

## Supported Integrations

Geckoboard connects to 80+ tools including:
- **Analytics**: Google Analytics, Mixpanel, Adobe Analytics

## Getting Started

1. **Connect Data Sources**: Link tools and platforms you use
2. **Choose Widgets**: Select pre-built widgets or create custom ones
3. **Arrange Dashboard**: Drag and drop widgets into layout
4. **Display**: Show on TV screens or share links with team
5. **Iterate**: Refine metrics based on team feedback

## Best Practices

- **Limit Metrics**: Show only the most important KPIs (5-8 per dashboard)
- **Use Color**: Green for positive, red for alerts, yellow for warnings
- **Update Frequency**: Set appropriate refresh rates for each metric
- **Team Input**: Involve team in deciding which metrics to track
- **Regular Reviews**: Periodically reassess which metrics matter most
