## Overview

Timetastic is online software specifically designed to help businesses manage and record employee absences. Operating since 2012, it provides an intuitive, visual approach to tracking time off, making it easy for both employees and managers.

## Key Features

### Wallchart View
The signature wallchart provides instant visibility into team absences with a visual calendar showing:
- Who's off and when
- Absence types (vacation, sick, etc.)
- Upcoming time off
- Team coverage at a glance

### Leave Request Workflow
Employees can:
- Request time off via web or mobile
- Check their remaining balance
- View their booking history
- See team calendar before requesting

Managers can:
- Approve or deny requests
- Check team coverage
- Set approval rules
- Receive instant notifications

### Automated Tracking
- Automatic absence calculations
- Real-time allowance updates
- Accrual tracking
- Carry-over management
- Anniversary date tracking

### Leave Types
Flexible configuration for all leave types:
- Annual leave / vacation
- Sick leave
- Maternity/Paternity
- Bereavement
- Compassionate leave
- Unpaid leave
- Custom leave types
