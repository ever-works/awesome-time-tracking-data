## Overview

StayFocusd is a productivity extension for Google Chrome that helps users stay focused on work by restricting the amount of time spent on time-wasting websites. The extension provides analytics to help understand and control internet usage.

## Core Features

### Time Tracking & Blocking
- Configure a blacklist of distracting sites
- Monitor time spent on blacklisted sites
- Automatically block sites once time limit expires
- Set specific daily time allowances per site or group

### Granular Control
Block at multiple levels:
- Entire sites
- Specific subdomains
- Specific paths/pages
- In-page content (videos, images, forms)

### Platform-Specific Blocking
Block features within platforms:
- YouTube videos
- TikTok content
- Instagram feeds
- Rather than blocking entire sites

## Scheduling Features

### Time Windows
- Enforce restrictions only during work hours (e.g., 9 AM - 6 PM)
- Configure specific days (Monday-Friday)
- Unrestricted access outside defined periods
- Customizable daily schedules

## Nuclear Option

### Maximum Focus Mode
The unique "Nuclear Option" feature:
- Blocks sites for set hours with NO cancel option
- Ensures unwavering focus during critical work
- Cannot be disabled once activated
- Perfect for deadline-driven work

## Time Allowances

### Flexible Allocation
- Set 10 minutes/day for Instagram
- Zero access to Twitter during work hours
- Different limits for different site groups
- Customize based on individual needs

## Analytics

Track and understand internet usage:
- Time spent on each site
- Daily usage patterns
- Productivity metrics
- Distraction identification

## Platform Availability

- Chrome Web Store
- Microsoft Edge Add-ons
- Average rating: 4.5 out of 5 stars

## Use Cases

### Work Focus
Block social media and news sites during work hours to maintain productivity.

### Study Sessions
Eliminate distractions during study time with strict blocking.

### Deadline Crunch
Use Nuclear Option to force focus during critical project periods.

### Habit Building
Gradually reduce time on distracting sites to build better habits.

## Key Advantage

Unlike simple blockers, StayFocusd combines time tracking with flexible, scheduled blocking and the enforced Nuclear Option, providing both gentle guidance and strict enforcement depending on user needs.
