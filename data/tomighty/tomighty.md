## Overview

Tomighty is a free and open-source desktop timer specifically designed for the Pomodoro Technique®. Created and developed by Célio Cidral Junior, it provides a simple, elegant solution for time management across all major operating systems.

## Platform Support

- **Windows**: Native application
- **macOS**: Native application
- **Linux**: Full support
- **Browser**: Web-based version available

## Technical Details

- Written in Java for cross-platform compatibility
- Runs as a native Java Swing application
- Lightweight: <12 MB RAM footprint
- Simple and unintrusive design
- Open-source (Apache License 2.0)
- Works identically across all platforms

## Key Features

- Desktop system tray integration
- Classic Pomodoro intervals (25/5/15)
- Customizable timer durations
- Audio notifications
- Visual alerts
- Minimal resource usage
- No internet connection required

## How It Works

1. Download and install Tomighty
2. The app sits in your system tray
3. Start a 25-minute Pomodoro
4. Get notified when time is up
5. Take a 5-minute break
6. Repeat the cycle
