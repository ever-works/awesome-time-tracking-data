## Overview

Jibble's Chrome extension brings time tracking directly into your browser, allowing you to track time on tasks and projects without leaving your current tab or workflow.

## Features

- One-click time tracking from browser
- Works across all websites and web apps
- Automatic activity detection
- Project and task selection
- Offline tracking capability
- Notes and descriptions
- Quick timer controls
- Seamless sync with Jibble platform
- Team time tracking
- Visual time reports
- GPS tracking for location verification
- Integration with popular project management tools

## Team Features

- Track team members' time
- Real-time activity monitoring
- Attendance tracking
- Timesheet approvals
- Payroll integration
- Leave management
- Schedule management

## Integration

- Google Calendar
- Slack
- Microsoft Teams
- Various payroll systems
- Export to Excel and CSV

## Platform Support

- Chrome browser extension
- Web application
- iOS and Android mobile apps
- Desktop applications
