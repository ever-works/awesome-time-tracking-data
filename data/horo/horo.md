## Overview

Horo is a streamlined timer application for macOS that provides quick countdown timers directly in the menu bar, perfect for Pomodoro technique, meeting timeboxes, and focused work sessions.

## Features

- **Natural Language Input**: Type timers in plain English like "25 minutes" or "1 hour"
- **Menu Bar Display**: Countdown visible in menu bar at all times
- **Multiple Timers**: Set multiple timers running simultaneously
- **Keyboard Shortcuts**: Global shortcuts to start, pause, and stop timers
- **Notifications**: Desktop notifications when timers complete
- **Pomodoro Support**: Pre-configured Pomodoro timer presets
- **Recent Timers**: Quick access to recently used timer durations
- **Custom Sounds**: Choose from multiple alert sounds or use system sounds
- **Dark Mode**: Full support for macOS dark mode
- **No Distractions**: Minimal interface that stays out of your way

## Pricing

Free version available with basic timer functionality. Pro version ($6.99 one-time) adds unlimited concurrent timers, custom sounds, and advanced preferences.
