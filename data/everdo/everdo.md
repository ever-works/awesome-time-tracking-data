## Overview

Everdo is a cross-platform GTD (Getting Things Done) application that prioritizes privacy and data security. Unlike cloud-based alternatives, Everdo stores all information locally on your device with end-to-end encryption, making it ideal for users who value data privacy and offline accessibility.

## Features

### Privacy & Security
- **Local Data Storage**: All information stays on your device, never sent to external servers
- **End-to-End Encryption**: Secure encryption protects your task data
- **Offline Availability**: Full functionality without internet connection
- **No Account Required**: Use without creating online accounts or sharing personal information
- **Data Ownership**: Complete control over your productivity data

### GTD Implementation
- **Inbox Processing**: Capture and process tasks following GTD methodology
- **Next Actions**: Clear list of immediately actionable items
- **Projects**: Multi-step outcomes organized with supporting actions
- **Contexts**: Filter tasks by location, tool, or situation (@home, @computer, etc.)
- **Someday/Maybe**: Store future possibilities without cluttering active lists
- **Waiting For**: Track delegated items and expected responses

### Task Management
- Quick capture with keyboard shortcuts
- Hierarchical project organization
- Due dates and scheduling
- Notes and attachments
- Tags and labels for flexible organization
- Focus mode for distraction-free work

### Platform Support
- **Linux**: Native application optimized for Linux distributions
- **Windows**: Full Windows desktop support
- **macOS**: Native Mac application
- **Cross-Platform Sync**: Optional encrypted sync between devices

## GTD Features

Everdo fully implements David Allen's Getting Things Done methodology:
- Capture: Quickly add tasks to inbox
- Clarify: Process inbox items into actionable tasks
- Organize: Categorize by project, context, and priority
- Reflect: Weekly review tools and reports
- Engage: Focus on next actions with context filtering

## Use Cases

- **Privacy-Conscious Professionals**: Users who need task management without cloud dependence
- **Offline Workers**: People who work in environments with limited or no internet
- **Security-Sensitive Roles**: Professionals handling confidential information
- **GTD Practitioners**: Users committed to David Allen's methodology
- **Linux Users**: Rare native Linux GTD application

## Target Audience

- Experienced GTD practitioners seeking privacy
- Security-conscious professionals
- Linux desktop users
- Anyone wanting offline-first task management
- Users who prefer local data storage over cloud services

## Philosophy

Everdo believes productivity software should respect user privacy and work independently of internet connectivity. The app demonstrates that powerful GTD implementation doesn't require sacrificing data ownership or privacy.
