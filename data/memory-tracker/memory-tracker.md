## Overview

Memory Tracker is Timely's proprietary AI-powered automatic time tracking technology that captures every minute spent in digital tools, then uses machine learning to draft timesheet entries for user review and approval.

## How It Works

1. **Background Capture**: Runs quietly tracking apps, websites, documents, meetings, emails used
2. **AI Processing**: Machine learning analyzes activity patterns
3. **Draft Timesheets**: AI suggests time entries based on captured data
4. **User Review**: Employee reviews and approves (or edits) suggested entries
5. **Final Timesheet**: Approved entries create accurate timesheet

## Key Features

- Tracks time in all web and desktop applications
- No manual timers to start/stop
- AI learns project patterns over time
- Complete timesheets in under 2 minutes daily
- Privacy-first: no keystroke logging or screenshots

## Privacy Approach

- All data private until user approves sharing
- No screenshots or keystroke monitoring
- Employee controls what gets shared
- Only approved entries visible to managers

## Benefits

- 100% time capture (never miss billable hours)
- Minimal user effort (AI does heavy lifting)
- Accurate timesheets without manual tracking
- Prevents revenue loss from untracked time
