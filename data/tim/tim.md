## Overview

Tim is a time tracker for macOS that features beautiful charts and a fully native Mac interface with complete light and dark mode support.

## Features

- Beautiful visual charts showing hours, averages, and revenue
- Fully native macOS UI
- Light and dark mode support
- Simple, intuitive time tracking
- Project and task organization
- Detailed reporting
- Export capabilities
- Menu bar integration
