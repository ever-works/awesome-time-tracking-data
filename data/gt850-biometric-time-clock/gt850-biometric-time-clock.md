## Overview

The GT850 is a premium biometric time clock that combines advanced facial recognition technology with a large, user-friendly touchscreen interface, designed for demanding work environments.

## Key Features

**Display**:
- 8-inch high-resolution touchscreen
- Full-color display
- Intuitive touch interface
- Easy-to-read in various lighting conditions

**Biometric Technology**:
- Integrated color camera
- Optimized facial recognition algorithms
- Liveness detection to prevent spoofing
- Fast recognition speed (typically <1 second)
- Works with safety equipment and PPE

**Clock-In Options**:
- Facial recognition (primary method)
- RFID badge backup
- PIN code entry
- Manual administrator override

## Applications

- Construction sites

- Field service operations
- Any environment needing reliable time capture

- Eliminates buddy punching

- Screen: 8" color touchscreen
