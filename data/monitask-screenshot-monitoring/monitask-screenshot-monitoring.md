## Overview

Monitask is an all-in-one employee monitoring solution featuring screenshots, activity tracking, and productivity reports. The screenshot feature helps track employees and their performance remotely by providing visual documentation of work.

## Screenshot Monitoring Features

### Automatic Capture
- Automatically capture and record real-time screenshots
- Configurable interval settings
- Default interval or custom frequency
- Screenshots taken only while actively tracking time

### Privacy Controls
- Screenshots can be turned off entirely
- Blurring option for sensitive content
- Only captures during manual clock-in periods
- No tracking outside work hours

### Transparency
Monitask is not spy software:
- Requires employee permission to install
- Only works when user is clocked in
- Employee knows when being monitored
- Tracking disabled outside work hours

## How It Works

1. Employee clocks in using desktop application
2. Screenshots captured at set intervals
3. Activity levels monitored during work time
4. Manager reviews screenshots with time entries
5. Employee clocks out to end monitoring
