## Overview

Time Sink is a macOS application designed to help you understand where your time goes while using your Mac. It automatically records how long each app and individual windows within it are open and in the foreground.

## Features

- **Automatic App & Window Tracking** - Tracks time spent in every app and window automatically
- **Pools** - Create groups of apps and windows to track related work together (e.g., all windows for a specific project)
- **Ad-Hoc Timers** - Automatically creates timers based on idle time and work sessions
- **Detailed Reports** - Always-updated usage reports with multiple sorting options
- **Time Period Filtering** - Easily change the time period displayed in reports
- **Export Capabilities** - Export data to text files for processing in other applications
- **No Manual Input** - Completely automatic tracking without needing to start/stop timers
- **Window-Level Precision** - Tracks individual windows within apps, not just applications

## Use Cases

Ideal for:
- Understanding personal productivity patterns
- Tracking time spent on different projects
- Answering "what did I do today?"
- Freelancers monitoring billable time informally
- Anyone curious about their Mac usage habits
