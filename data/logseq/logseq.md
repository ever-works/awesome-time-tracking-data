## Overview

Logseq is a privacy-first, open-source knowledge base that combines personal knowledge management with task tracking and time recording capabilities.

## Time Tracking Features

### Built-in Time Tracking

Logseq has a built-in feature to track the time spent DOING a TODO block. When you complete tasks, Logseq can automatically record completion times.

### Time Tracking Plugins

**Logseq Time Tracker Plugin:**
- Tracks time spent on individual tasks
- Integrates with TODO blocks
- Available on GitHub (vyleung/logseq-time-tracker-plugin)

**Logseq Time Recorder:**
- Simple plugin for tracking working hours
- Clock in/clock out button functionality
- Manually editable time records
- Functions as a punch clock for work sessions

**Logseq DONE Time Plugin:**
- Records when tasks are completed
- Useful for standups and meetings
- Available in Logseq Marketplace
- Tracks task completion timestamps

## Knowledge Management Integration

### Task Management

Logseq excels at personal knowledge management and task tracking with features including:
- Searching for tasks across all notes
- Adding new to-dos with inline commands
- Filtering results by marker or priority
- Linking tasks to knowledge base entries

### Daily Notes

Integration with daily notes allows for:
- Time tracking within daily journal entries
- Task completion records
- Work session documentation
- Temporal organization of knowledge

## Plugin Ecosystem

### Productivity Plugins

Logseq's plugin ecosystem includes:
- Task completion trackers
- Time recording tools
- TODO list management
- Search and organization tools

All plugins can be installed directly from the Logseq Marketplace within the app.

## Key Benefits

- **Privacy-first**: All data stored locally
- **Open-source**: Transparent development and community-driven
- **Flexible**: Highly customizable with plugins
- **Integrated**: Combines knowledge management with time tracking
- **Free**: No subscription required

## Target Users

Knowledge workers, researchers, students, developers, and anyone who wants to combine personal knowledge management with time tracking in a privacy-focused, open-source environment.

## Platform Availability

Desktop (Windows, Mac, Linux) and Mobile (iOS, Android)
