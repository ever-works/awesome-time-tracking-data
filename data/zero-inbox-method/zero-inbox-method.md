## Core Principle

Process every email to zero (or near-zero) regularly by making quick decisions rather than using inbox as to-do list.

## The Process

For each email, immediately:

1. **Delete**: If not important
2. **Delegate**: Forward to appropriate person
3. **Respond**: If takes under 2 minutes
4. **Defer**: Move to task list with specific action
5. **Archive**: File for reference

## Benefits

- Reduced email stress
- Nothing lost or forgotten
- Clear mind
- Faster response times
- Better prioritization
- Inbox as processing tool, not storage

## Implementation

### Set Processing Times

Schedule 2-3 specific times daily for email.

### Turn Off Notifications

Prevent constant interruptions.

### Use Filters and Rules

Automate routine sorting.

### Unsubscribe Aggressively

Reduce incoming volume.

### Touch Each Email Once

Make decision on first read.

## Common Challenges

### High Volume

Start with last 24 hours, declare email bankruptcy on rest.

### Decision Fatigue

Use templates for common responses.

### Requires Action

Move to task manager, not back to inbox.

## Maintenance

- Process to zero daily
- Review weekly for missed items
- Adjust filters as needed
- Regular unsubscribe sessions
