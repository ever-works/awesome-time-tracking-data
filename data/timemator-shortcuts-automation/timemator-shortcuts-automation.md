## Overview

Timemator is an automatic time tracking app for Mac and iPhone that uses intelligent rules to automatically track time based on your activities, eliminating the need to manually start and stop timers.

## Three-Pronged Tracking

### 1. Automatic Activity Timeline
Timemator automatically creates a timeline as you work, capturing what apps and files you use throughout the day.

### 2. Rule-Based Auto-Recording
Set up rules to automatically record time when:
- VS Code + specific project folder is open
- Particular website URL is active
- Specific app is in use
- Certain file types are accessed

### 3. Manual Timer
Manual timer available for one-off tasks that don't fit automatic rules.

## Automation Examples

- Auto-track "Client A" project when working in designated folder
- Start "Design" timer when Figma is active
- Track "Writing" when specific documents are open
- Record "Research" time on certain websites

## Privacy & Offline

- Works completely offline
- Respects your privacy
- No data sent anywhere
- All tracking stays on device

## Platform Support

- macOS
- iOS (iPhone and iPad)
- Sync between devices

## Use Cases

- Freelancers tracking billable hours
- Developers logging project time
- Knowledge workers analyzing time allocation
- Anyone who forgets to use manual timers

## Key Advantage

Eliminates the burden of remember to start/stop timers by automatically detecting project context and recording time without user intervention.
