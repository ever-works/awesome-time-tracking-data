## Overview

TomatoBar is a minimal, elegant Pomodoro timer designed specifically for macOS that lives discretely in your menu bar. It provides a clean, unobtrusive way to practice the Pomodoro Technique on your Mac.

## Features

- Configurable work and rest intervals
- Optional sounds for timer notifications
- Discreet actionable notifications
- Global hotkey support for quick access
- Lives in the macOS menu bar for easy access
- Minimal and elegant design
- Open-source on GitHub

## How It Works

TomatoBar sits in your macOS menu bar, allowing you to quickly start, stop, and monitor Pomodoro sessions. The timer countdown is visible at a glance in your menu bar, and you can customize the duration of both work sessions and breaks to fit your workflow.

## Technical Details

- Platform: macOS only
- Type: Native macOS application
- License: Open-source
- Repository: Available on GitHub

## Ideal For

Mac users who want a simple, distraction-free Pomodoro timer that integrates seamlessly with macOS without requiring a separate window or complex interface.
