## Parkinson's Law

"Work expands to fill the time available for its completion." - Cyril Northcote Parkinson

## Application

Set tighter, artificial deadlines to:

- Create healthy urgency
- Prevent perfectionism
- Reduce overthinking
- Improve focus
- Increase completion rate

## Strategies

### Cut Time Estimates in Half

If you think a task takes 4 hours, allocate 2 hours.

### Set Multiple Milestones

Break projects into shorter deadline chunks.

### Use Timeboxing

Allocate fixed time regardless of completion.

### Public Commitments

Share deadlines with others for accountability.

### Buffer Real Deadlines

Set personal deadline 1-2 days before actual deadline.

## Benefits

- Reduces procrastination
- Eliminates unnecessary polish
- Forces prioritization
- Improves time estimation
- Creates momentum

## Cautions

- Don't sacrifice quality unnecessarily
- Allow appropriate time for complex work
- Build in review time
- Consider stress levels
