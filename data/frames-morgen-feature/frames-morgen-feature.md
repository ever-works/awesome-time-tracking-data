## Overview

Frames is an innovative feature in Morgen that revolutionizes time blocking by creating recurring time blocks with intelligent task filtering. A Frame is a designated time period where specific types of tasks can be automatically scheduled based on customizable criteria.

## How Frames Work

### Recurring Time Blocks
Frames allow users to establish recurring time blocks in their calendar for specific types of work, such as:
- Morning deep work sessions
- Afternoon administrative tasks
- Evening planning and review
- Weekly strategic thinking time
- Daily communication windows

### Intelligent Filtering
Each Frame uses filters to determine which tasks can appear within it, based on:
- **Task Source**: Which app or system the task comes from
- **Task List**: Specific lists or projects
- **Task Size**: Estimated duration or complexity
- **Tags**: Custom labels and categories
- **Combinations**: Multiple criteria working together

## Key Benefits

### Automated Task Scheduling
- Tasks automatically flow into appropriate Frames
- Reduces manual scheduling decisions
- Ensures tasks are scheduled during suitable time blocks
- Maintains work-life boundaries through Frame constraints

### Contextual Work Organization
- Group similar work types into dedicated time blocks
- Reduce context switching by batching task types
- Optimize energy allocation across different work types
- Protect specific hours for specific work categories

### Flexible Customization
- Define Frame rules based on your unique workflow
- Adjust Frame sizes and frequencies as needed
- Combine multiple filters for precise task placement
- Override automatic placement when necessary

## Example Frames

### Deep Work Frame
- **Time**: Monday-Friday, 9:00 AM - 11:30 AM
- **Filters**: Tasks from project management, tagged "focus", size > 30 minutes
- **Purpose**: Protected time for complex, concentration-intensive work

### Administrative Frame
- **Time**: Monday-Friday, 2:00 PM - 3:00 PM
- **Filters**: Tasks tagged "admin" or "email", size < 30 minutes
- **Purpose**: Batch processing of routine tasks

### Planning Frame
- **Time**: Sunday, 6:00 PM - 7:00 PM
- **Filters**: Tasks from personal lists, tagged "planning"
- **Purpose**: Weekly review and upcoming week preparation

## Integration with AI Planner

Frames work in conjunction with Morgen's AI Planner:
- AI suggests tasks for each Frame based on filter criteria
- Users preview and approve AI suggestions
- AI learns from user adjustments to improve future suggestions
- Frames provide structure while AI provides intelligence

## Best Practices

- Start with 2-3 Frames and expand as needed
- Align Frames with your biological prime time
- Use specific filters to prevent Frame overlap
- Review Frame effectiveness weekly and adjust
- Protect your most important work with dedicated Frames
- Allow flexibility for urgent tasks outside Frame constraints
