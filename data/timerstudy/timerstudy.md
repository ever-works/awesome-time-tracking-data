## Overview

Timer.Study is a free Pomodoro timer app featuring task tracking, customizable sessions, and statistics. Designed specifically for students, it requires no sign-up and saves everything in browser local storage.

## Task Tracking System

### Add Tasks
- Create tasks you want to work on
- Click task to make it active focus
- Time tracked on active task
- Optional task system (can use without tasks)

### Visual Progress
- Each filled dot = one completed Pomodoro
- Total focus time displayed (e.g., "45m")
- Visual representation of effort
- Daily achievement tracking

### Task Management
- Add multiple tasks
- Set active task
- Track time per task
- Monitor overall progress

## Statistics & Data

### Local Storage
- Everything saved in browser
- Tasks persist between sessions
- Settings remembered
- Stats tracked daily

### Daily Reset
- Statistics reset at midnight
- Fresh start each day
- Daily progress tracking
- Consistent metrics
