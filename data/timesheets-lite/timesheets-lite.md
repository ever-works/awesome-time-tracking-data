## Overview

Timesheets Lite is a simple to use timesheet application designed for project-based businesses, consultants, lawyers, builders, accountants, programmers, and engineering firms.

## Key Features

### Free for Small Teams
- Completely free for single connections (up to 3 users)
- Simple to use interface
- Uses Microsoft Access database

### Time Tracking Capabilities
- Create and delete employees and projects
- Track time against common tasks in projects
- Task Timer functionality for accurate time capture

### Reporting
- Generate reports in multiple formats:
  - Microsoft Excel
  - Plain text
  - HTML

### Security & Control
- Multi-user system with user security
- Password control for data protection

## Ideal Use Cases

Timesheets Lite is perfect for:
- Project based businesses
- Consultants
- Lawyers
- Builders
- Accountants
- Programmers
- Engineering firms
