## Overview

The **Overtime Calculator** by busybusy is a web-based tool for estimating overtime pay using tracked work hours. It supports both daily and weekly overtime rules and allows customization of thresholds and multipliers to match different policies and common federal overtime guidelines.

## Features

- **Web-based overtime calculator** accessible via browser
- **Two calculation modes**:
  - **Classic mode**: Simple weekly overtime calculations
  - **Advanced mode**: More detailed control over overtime rules
- **Customizable overtime rules (Advanced mode)**:
  - Set weekly hours threshold for when overtime begins
  - Set base overtime pay multiplier (e.g., 1.5x)
  - Optional daily hours threshold for daily overtime
  - Optional daily overtime pay multiplier separate from weekly rules
- **Input options**:
  - Regular hourly wage
  - Start day of the workweek
  - Total hours worked for each day of the week
- **Output details**:
  - Breakdown of **regular hours** and **overtime hours**
  - **Daily** and **weekly** totals
  - Calculated **regular earnings**, **overtime earnings**, and **total pay**
  - Results displayed in an organized summary table
- **Guidance on overtime concepts**:
  - Explanation of thresholds (hours before overtime applies)
  - Example walkthrough for calculating overtime manually
- **General compliance orientation**:
  - Designed around common federal overtime guidelines
  - Flexible enough to adjust for different policies and local rules
- **Use cases**:
  - Contractors with variable daily/weekly hours
  - Employers or payroll admins checking overtime scenarios
  - Employees estimating expected overtime pay
- **Disclaimer**:
  - Intended for informational/estimating purposes only
  - Users are responsible for confirming results and complying with local/state laws and regulations
