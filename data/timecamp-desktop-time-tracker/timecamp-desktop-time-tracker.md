## Overview

TimeCamp is an automatic time tracking application that runs on desktop computers to monitor application usage, websites visited, and document activity. It combines automatic tracking with manual time entry capabilities, offering comprehensive time analytics for individuals and teams.

## Key Features

- **Automatic Tracking**: Monitors all computer activity automatically
- **App & Website Tracking**: Records time spent in applications and websites
- **Keyword Detection**: Categorizes time based on document names and content
- **Manual Time Entry**: Option to add time entries manually
- **Project Management Integration**: Syncs with Asana, Trello, Jira, and 50+ tools
- **Timesheets**: Generate professional timesheets
- **Invoicing**: Create invoices from tracked time
- **Budgeting**: Set project budgets and track spending
- **Productivity Reports**: Detailed analytics on time usage
- **Team Management**: Track and manage team time
- **Mobile Apps**: iOS and Android companions
- **Attendance Tracking**: Optional clock in/out features

## Automatic Categorization

- AI-powered project detection

## Privacy Features

- Optional private/incognito mode
