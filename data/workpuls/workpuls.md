## Overview

Workpuls is an employee monitoring and automatic time tracking platform that provides detailed insights into team productivity. It tracks application usage, website visits, and active time to help managers understand how work hours are spent.

## Features

- Automatic time tracking
- Application and website monitoring
- Screenshot capture (optional)
- Activity levels tracking
- Productivity categorization
- Real-time dashboards
- Detailed reports and analytics
- Project time allocation
- Attendance tracking
- Stealth mode option
- Customizable productivity rules
- Integration with project management tools

## Pricing

Workpuls offers tiered pricing based on features:
- Basic plan with core tracking
- Premium plan with advanced analytics
- Enterprise plan with custom features

Pricing is per user per month, with discounts for annual billing.
