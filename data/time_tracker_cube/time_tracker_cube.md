## time_tracker_cube

### Overview
An Arduino-based physical time tracking cube that integrates with the Toggl Track API. Each side of the paper cube is associated with a specific task, and placing a side facing upwards starts tracking time for that task.

**Category:** Others  
**Website:** https://time.is/

### Features
- **Arduino-based device** for controlling and detecting cube orientation.
- **Physical paper cube form factor**, intended to sit on a desk.
- **Six configurable sides**, each corresponding to a different task or activity.
- **Automatic task selection** based on which side is facing upwards.
- **Toggl Track API integration** to start tracking time for the selected task.
- **Hands-on interaction**: switch tasks by physically rotating the cube.
