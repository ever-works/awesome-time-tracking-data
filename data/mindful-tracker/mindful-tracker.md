## Overview

Mindful Tracker is designed with a single purpose: to time unguided meditation sessions and save them as "Mindful Minutes" in the Apple Health App. It's a minimalist tool for those who practice meditation without guidance.

## Features

- Set a timer for specific meditation durations
- Start a stopwatch for open-ended sessions
- Automatic saving to Apple Health as Mindful Minutes
- Clean, distraction-free interface
- Quick session start with minimal taps
- Integration with iOS Health app

## Use Case

Ideal for meditators who:
- Practice unguided meditation
- Want to track mindfulness in Apple Health
- Prefer simplicity over feature-rich apps
- Track wellness metrics across iOS devices
