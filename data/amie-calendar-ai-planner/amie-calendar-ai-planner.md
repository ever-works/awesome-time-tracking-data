## Overview

Amie is an all-in-one AI-powered productivity app that combines calendar management, task tracking, and email integration with intelligent scheduling assistance.

## AI Features

### AI Chat Integration
- Understands natural language instructions
- Automatically schedules tasks with time blocking
- Example: "Do house repairs on Sunday" → schedules with 2-hour time block
- Smart task duration estimation

### Natural Language Processing
- Add events by typing phrases like "Go to the grocery store @this Sunday at 3:00 p.m."
- Automatically extracts time, date, and details
- Creates time blocks on calendar automatically

## Key Features

### Time Blocking
- Assign specific time slots for each task
- Work within fixed timeboxes instead of open-ended tasks
- Visual calendar representation of time allocation
- Drag-and-drop time block adjustments

### Email Integration
- Turn emails into tasks
- Schedule email-derived tasks directly on calendar
- Integrated email management within app

### Task Management
- Comprehensive to-do list system
- Integration with calendar for unified view
- Time-boxed task approach
- Project organization
