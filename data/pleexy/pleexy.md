## Overview

Pleexy is a specialized integration platform that enables seamless 2-way task synchronization between Notion and Todoist, allowing users to maintain task consistency across both popular productivity platforms without manual copying or reconciliation.

## Core Functionality

### 2-Way Synchronization

**Bidirectional Sync**:
- Tasks created in Notion appear in Todoist
- Tasks created in Todoist sync to Notion
- Updates in either platform reflect in the other
- Deletions propagate across systems
- Real-time or scheduled sync options

**What Gets Synced**:
- Task titles and descriptions
- Due dates and deadlines
- Priority levels
- Tags and labels
- Completion status
- Updates and modifications

## Limitations & Considerations

### What Doesn't Sync

**Platform-Specific Features**:
- Notion: Rich text formatting, databases, relations
- Todoist: Sub-tasks depth, karma points
- Some features unique to each platform
- May require manual handling

**Technical Constraints**:
- API rate limits
- Sync delay (not instant)
- Complex database structures
- Custom field limitations

### Best Practices for Limitations

**Workarounds**:
- Keep synced data simple
- Use platform-specific features separately
- Document what doesn't sync
- Train users on limitations
