## Overview

Time category templates are pre-configured time entry shortcuts that capture common activity patterns, allowing users to log frequent tasks with a single click rather than manually entering all details each time.

## Template Components

### Pre-filled Fields
- Client/project assignment
- Task or activity type
- Category (billable/non-billable)
- Default duration (if applicable)
- Notes template
- Billing rate

### One-Click Entry
- Select template from list
- Adjust duration if needed
- Add specific notes if required
- Save—complete in seconds

## Types of Templates

### Recurring Meetings
- Weekly team standup
- Monthly client status calls
- Quarterly planning sessions
- Regular 1:1s

### Routine Activities
- Email/communication
- Code review
- Documentation
- Research
- Administrative tasks

### Project-Specific Work
- [Client] - Development
- [Client] - Design
- [Client] - Project Management
- [Client] - Client Communication

## Benefits

### Speed
- Reduces entry time from 60+ seconds to 5-10 seconds
- Enables sub-2-minute daily time entry
- Eliminates typing and selection fatigue
- Supports quick mobile entry

### Accuracy
- Consistent categorization
- Correct billing rates applied
- Proper project association
- Reduces entry errors

### Completeness
- Low friction encourages logging small increments
- Quick entry means less forgotten time
- Easier to track interruption-heavy days

## Implementation

### Template Creation
1. Analyze past 30 days of time entries
2. Identify activities logged 5+ times
3. Create template for each frequent activity
4. Test with users and refine
5. Roll out to team

### Naming Conventions
- Clear, descriptive names
- Include client/project for context
- Use consistent format
- Group related templates

Example:
- "Acme Corp - Development"
- "Acme Corp - Meetings"
- "Acme Corp - Email"
- "Internal - Code Review"

### User Adoption
- Train team on template usage
- Demonstrate time savings
- Encourage creation of personal templates
- Share most-used templates across team

## Best Practices

- Create 10-15 templates covering 80% of work
- Review and update templates quarterly
- Allow users to create personal templates
- Standardize templates for shared activities
- Make templates prominent in UI

## Advanced Features

### Smart Suggestions
- System suggests relevant templates based on time/day
- Machine learning predicts likely template
- Recent templates prioritized

### Template Inheritance
- Project templates auto-available to project team
- Department-wide templates
- Individual customization allowed

### Template Analytics
- Track template usage frequency
- Identify gaps (high manual entry activities)
- Optimize template library

## Impact on Two-Minute Rule

Templates are crucial for achieving sub-two-minute daily time entry:
- Without templates: 2-3 minutes per entry × 8 entries = 16-24 minutes
- With templates: 10-15 seconds per entry × 8 entries = 1.5-2 minutes

This difference determines whether time tracking becomes a sustainable habit or fails due to friction.
