## Overview

Task Code Time Allocation provides granular visibility into how work time is spent across different activity types, enabling data-driven decisions about process optimization and resource allocation.

## Common Task Code Categories

### Billable/Direct Work
- Development/Production
- Client Services
- Consulting/Advisory
- Project Delivery

### Support Activities
- Meetings (internal)
- Meetings (client)
- Email/Communication
- Documentation
- Code Review
- Testing/QA

### Administrative
- Timesheets/Admin
- Training (receiving)
- Training (delivering)
- HR activities

### Business Development
- Proposals
- Sales calls
- Marketing
- Networking
