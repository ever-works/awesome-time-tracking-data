## Overview

Timing is an automatic time tracking application exclusively for macOS that captures all computer activity in the background, eliminating the need for manual timers. Using intelligent categorization and project detection, it's particularly popular among developers, designers, and knowledge workers who need accurate time records without workflow interruption.

## Key Features

- **Fully Automatic**: Tracks all application usage, documents, and websites automatically
- **AI Categorization**: Machine learning suggests project assignments based on activity patterns
- **Privacy-First**: All data stored locally on Mac, never uploaded to cloud
- **Developer-Friendly**: Built-in APIs through AppleScript and Web API
- **Project Detection**: Automatically recognizes which project you're working on based on context
- **Timeline View**: Visual timeline of your entire day's activities
- **Productivity Insights**: Analyze how time is spent across apps and projects
- **Offline Capable**: Works without internet connection
- **No Manual Timers**: Never need to remember to start/stop tracking
- **Native Mac App**: Optimized for macOS with full system integration
