# TrackingTime Academy

## Overview
TrackingTime Academy is an educational resource from TrackingTime that helps individuals and teams learn how to use time tracking effectively within the TrackingTime ecosystem. It offers role-specific guides, implementation frameworks, and video-based tutorials focused on practical, day-to-day use cases such as productivity, client billing, and payroll.

## Features
- **Role-based learning paths**
  - Guidance tailored to different work roles and responsibilities.
  - Examples of covered roles:
    - Team leads
    - HR managers
    - Freelancers
    - Finance leads

- **Implementation guides for time tracking strategies**
  - Structured guidance on how to implement time tracking in an organization.
  - Separate best-practice guides for:
    - **Employees**: how to track hours consistently and accurately.
    - **Employers**: how to introduce time tracking, set expectations, and support adoption.

- **Practical use-case education**
  - Methodologies and recommendations for:
    - Achieving productivity goals using time tracking data.
    - Reporting and billing clients based on tracked time.
    - Simplifying payroll processes using TrackingTime.

- **Video learning hub**
  - Video tutorials designed to help users adopt time tracking as a habit.
  - Coverage of specific topics such as:
    - Time tracking policies
    - Using timesheets
    - Setting and using reminders
    - Using the TrackingTime browser extension
    - Tracking and reviewing hours
    - Setting up and using custom fields
    - Managing rates and costs
    - Tracking and managing time off

- **Integrations and workflow guidance**
  - Educational content on integrating TrackingTime into existing workflows.
  - Focus on using time tracking alongside other tools via integrations.
