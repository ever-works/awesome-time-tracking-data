## Overview

Keyboard shortcuts for time tracking enable experienced users to log time entirely via keyboard commands, eliminating mouse clicks and navigation that slow down entry, critical for achieving sub-two-minute daily time logging.

## Common Shortcuts

### Entry Creation
- `N` or `Ctrl/Cmd + N`: New time entry
- `T`: Start/stop timer
- `Tab`: Navigate between fields
- `Enter`: Save entry
- `Esc`: Cancel

### Navigation
- `↑/↓`: Move between entries
- `Ctrl/Cmd + D`: Duplicate last entry
- `Ctrl/Cmd + E`: Edit selected entry
- `Delete`: Remove entry

### Quick Actions
- `Ctrl/Cmd + S`: Save all entries
- `Ctrl/Cmd + F`: Search entries
- `/`: Focus search box
- `?`: Show shortcuts help

## Power User Benefits

- 5-10 second entry time (vs 30-60 seconds)
- No context switching to mouse
- Maintains coding/writing flow
- Reduced friction = better compliance
- Can log time while on calls

## Adoption Strategy

- Include shortcut cheat sheet in training
- Display shortcuts in UI tooltips
- Offer "power user" mode
- Celebrate fast entry times

## Impact on Two-Minute Rule

Keyboard shortcuts can reduce daily time entry from 3-5 minutes to under 1 minute for power users, making time tracking nearly invisible in daily workflow.
