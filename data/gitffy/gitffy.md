## Overview

Gitffy is a unique habit tracking application built for developers that leverages GitHub commit data to automatically track coding habits, productivity patterns, and development consistency without manual time logging.

## Features

- **GitHub Integration**: Automatically tracks habits based on GitHub commit activity
- **Commit Streaks**: Build and maintain coding streaks based on daily commits
- **Repository Insights**: See which repositories you're most active in
- **Coding Patterns**: Understand your coding patterns and most productive times
- **Language Tracking**: Track which programming languages you're using over time
- **Contribution Heatmap**: Visualize your coding activity with contribution heatmaps
- **Goals**: Set coding goals and track progress toward them
- **Habit Building**: Use commit activity to build consistent coding habits
- **Privacy-Focused**: Only accesses publicly available GitHub data or authorized repos
- **Developer Motivation**: Gamify your development work with achievements and milestones

## Pricing

Free to use with GitHub account authentication. No premium tier as it leverages public GitHub data for tracking.
