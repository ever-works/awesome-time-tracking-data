## Overview

Paymo is an integrated work management platform designed for small businesses and creative teams, combining project planning, time tracking, resource scheduling, and invoicing in a single, affordable solution.

## Features

- **Project Planning**: Gantt charts, task lists, and kanban boards for project visualization
- **Time Tracking**: Automatic and manual time tracking with desktop, mobile, and web timers
- **Task Management**: Create tasks with subtasks, dependencies, and assignments
- **Resource Scheduling**: Visual team calendar showing who's working on what
- **File Proofing**: Collect client feedback on files with markup and annotation tools
- **Invoicing**: Generate invoices from tracked time and expenses
- **Expense Tracking**: Log project expenses and attach receipts
- **Team Collaboration**: Comments, file sharing, and task discussions
- **Reports**: Profitability reports, time reports, and productivity analytics
- **Integrations**: Connects with Google Calendar, Zapier, Slack, and accounting software

## Pricing

Free plan for 1 user with basic features. Small Office at $5.95 per user per month. Business plan at $11.95 per user per month with all features including resource scheduling and unlimited storage.
