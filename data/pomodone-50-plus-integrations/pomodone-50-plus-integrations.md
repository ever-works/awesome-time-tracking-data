## How Integration Works

### Task Import
1. Connect PomoDone to your tools
2. Tasks automatically sync
3. Select task to work on
4. Start Pomodoro timer
5. Time tracked against original task

### Bidirectional Sync
- Task completion status syncs back
- Time entries update in source tool
- Comments and notes preserved
- Changes reflected in both systems
