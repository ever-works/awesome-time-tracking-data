## Overview

The Time Block Planner is a physical planning system created by productivity author Cal Newport (Deep Work, Digital Minimalism). The planner is specifically designed to support the time blocking method, where you plan your day in advance by assigning every minute to a specific activity.

## Features

- **Daily Time Block Pages**: Grid format for scheduling every hour of the day
- **Weekly Planning Sections**: Set priorities and goals for the week ahead
- **Metric Tracking**: Space to track key metrics and habits
- **Reflection Prompts**: End-of-day reflection on what worked and what didn't
- **Shutdown Ritual**: Structured end-of-workday routine
- **Quarterly Planning**: Long-term goal setting and project planning
- **Undated**: Start any time without wasted pages
- **Quality Construction**: Durable binding, premium paper

## Time Blocking Method

The planner supports Cal Newport's time blocking approach:
1. **Plan the Night Before**: Block out your next day in 30-minute increments
2. **Be Realistic**: Account for meetings, breaks, transitions
3. **Allow Flexibility**: Use conditional blocks for uncertain tasks
4. **Revise as Needed**: Draw new blocks when plans change
5. **Track Results**: Note what you actually accomplished
6. **Reflect Daily**: Learn from execution to improve future planning

## Benefits

- **Intentional Days**: Live according to plan, not reaction
- **Shallow Work Containment**: Limit low-value tasks to specific blocks
- **Deep Work Protection**: Guard time for important, cognitively demanding work
- **Reality Check**: See how you actually spend time vs how you plan
- **Reduced Decisions**: Eliminate constant "what should I do now" questions
- **Analog Focus**: Physical planner reduces digital distractions
