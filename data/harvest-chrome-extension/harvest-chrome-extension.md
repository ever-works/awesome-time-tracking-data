## Overview

The Harvest Chrome Extension places a convenient timer directly in the Chrome toolbar and integrates Harvest timers into popular project management tools for seamless time tracking.

## Features

### Browser Integration
- **Toolbar Timer**: Quick-access timer icon in Chrome toolbar
- **One-Click Start**: Begin tracking with a single click
- **Running Timer Display**: See active timer without opening full app
- **Timer Controls**: Start, stop, and edit directly from extension

### App Integrations
When installed, the extension adds Harvest timers to:

- **Asana**: Track time on Asana tasks without leaving the platform
- **Basecamp**: Timer buttons appear within Basecamp to-dos and projects
- **Trello**: Add time tracking to Trello cards
- **GitHub**: Track time spent on GitHub issues and pull requests

### Time Entry Management
- **Quick Entry**: Add manual time entries through the extension
- **Project Selection**: Choose from your Harvest projects and tasks
- **Notes**: Add descriptions to time entries
- **Edit Existing**: Modify recent time entries

### Synchronization
- **Real-Time Sync**: All entries sync instantly with Harvest web and mobile apps
- **Cross-Platform**: Works alongside Harvest desktop and mobile applications
- **Automatic Updates**: Running timers update across all devices

## Benefits

- **Reduced Context Switching**: Track time without leaving your work tools
- **Improved Accuracy**: Log time as you work rather than reconstructing later
- **Seamless Workflow**: Timers integrated into existing project management tools
- **Always Visible**: Toolbar icon reminds you to track time

## Supported Tools

- Asana
- Basecamp (all versions)
- Trello
- GitHub
- Gmail
- Google Calendar

## Requirements

- Active Harvest account (free trial or paid plan)
- Google Chrome browser
- Works with Chromium-based browsers
