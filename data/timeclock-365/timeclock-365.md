## Overview

TimeClock 365 is time tracking software designed specifically for construction and field service companies that use Microsoft 365. It integrates seamlessly with Microsoft Teams, SharePoint, and other Microsoft tools.

## Features

- Native Microsoft 365 integration
- Mobile time clock with GPS
- Geofencing capabilities
- Job and task tracking
- Time approval workflows in Teams
- Offline time tracking
- Photo and note capture
- Real-time reporting
- Export to payroll systems
- Cost code tracking
- Equipment time logging
- Integration with Microsoft Power BI for analytics

## Pricing

Subscription-based pricing, typically billed per user per month. Integrates with existing Microsoft 365 subscriptions.
