# Slack Time Tracking (TrackingTime Integration)

## Overview
Slack Time Tracking by TrackingTime lets you track work hours directly inside Slack. It connects Slack conversations and workflows with TrackingTime’s time tracking, timesheets, and analytics so teams can log time without leaving Slack.

## Features
- **Track time from within Slack** using TrackingTime commands so you can start tracking work right where conversations happen.
- **Automatic timesheets** generated from your tracked time in Slack.
- **Detailed analytics and reporting** based on time entries started or managed from Slack.
- **Minimal setup**: add TrackingTime from Slack’s Apps section and connect it to your TrackingTime account.
- **Browser extension support** (Chrome, Firefox, Safari, Edge) to keep track of work done in Slack via the TrackingTime extension.
- **Works as part of TrackingTime’s broader platform**, including:
  - Time tracking across projects and tasks
  - Project monitoring
  - Compliance with internal and external requirements
  - Flexible reporting on tracked time

## How It Works
1. **Sign up for TrackingTime** to create your workspace.
2. **Add TrackingTime to Slack** via Slack’s Apps directory and complete the installation prompts.
3. **Use TrackingTime commands in Slack** to start and manage time tracking while you work in channels and direct messages.

## Integrations
- **Slack**: Native integration to track time from Slack conversations using TrackingTime.
- **Other tools**: TrackingTime can sync with various apps your team uses daily so they can track time without switching between tools (specific apps not detailed on this page).
