## Overview

Apploye's Chrome time tracking extension combines traditional time logging with screenshot monitoring and productivity analysis, designed for teams that need detailed activity verification.

## Features

### Time Tracking
- **Browser-Based Timer**: Track time directly from Chrome
- **Automatic Tracking**: Optional automatic time capture
- **Manual Entry**: Add time retrospectively when needed
- **Project Assignment**: Categorize time by projects and tasks

### Activity Monitoring
- **Screenshot Capture**: Periodic screenshots during work sessions
- **Activity Level Tracking**: Monitor keyboard and mouse activity
- **App & URL Tracking**: Track which applications and websites are used
- **Idle Time Detection**: Automatically detect and handle idle periods

### Productivity Analysis
- **Productivity Scores**: Calculate productivity based on activity
- **Time Utilization**: See how time is distributed across tasks
- **Detailed Reports**: Comprehensive activity and time reports
- **Team Insights**: Manager view of team productivity

### Management Features
- **Timesheet Generation**: Automatic timesheet compilation
- **Approval Workflows**: Submit and approve timesheets
- **Project Budgets**: Monitor time against project budgets
- **Billing Integration**: Convert tracked time to invoices

## Use Cases

- Remote team management
- Client billing verification
- Employee productivity monitoring
- Project time allocation tracking

## Privacy Considerations

- Screenshot frequency is configurable
- Employees can see what's being tracked
- Blur sensitive information options
- Compliant with workplace monitoring regulations

## Compatibility

- Google Chrome
- Chromium-based browsers
