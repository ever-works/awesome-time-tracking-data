## Overview

Window Focus highlights the current working window and dims all others in the background, helping you concentrate on tasks at hand on your Mac.

## Key Features

### Window Highlighting
- Highlights the current app window that you have open
- Dims all apps and windows in the background
- Clear visual distinction between active and inactive windows

### Customization Options
- Adjust overlay tint color to any color of your choice
- Control overlay opacity level using an adjuster scale
- Fine-tune dimming effects to your preference
- Disable dimming for specific apps via app exceptions in preferences

### Menubar Integration
- Functions as a menubar app in the top menubar
- Easy access to app settings and controls
- Unobtrusive presence when not needed

## System Requirements

macOS 10.8 or later

## Use Cases

- Reducing distractions from multiple open windows
- Boosting productivity by maintaining focus
- Making active window more visually prominent
- Managing attention in complex workflows
