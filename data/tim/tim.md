## Overview

Tim is a simple time tracker that shows time right in the menu bar and can be controlled with keyboard shortcuts.

## Key Features

- **Menu Bar Integration**: Display current task and running timer directly in the macOS menu bar
- **Keyboard Control**: Operate entirely via keyboard shortcuts without touching the mouse
- **Task Visibility**: Hover over the timer in the bar to see a tooltip with the current task
- **Minimal Design**: Clean, uncluttered interface that stays out of your way

## Platform

Exclusively available for macOS.

## Use Case

Ideal for Mac users who prefer a lightweight, keyboard-driven time tracking solution without complex features or heavy system resources.
