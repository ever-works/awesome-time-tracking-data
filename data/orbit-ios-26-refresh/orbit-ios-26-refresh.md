## iOS 26 Update Highlights

Orbit received a comprehensive visual and functional refresh for iOS 26, bringing the app in line with Apple's latest design language and adding new AI capabilities.

## Visual Improvements

- Refreshed button designs
- Updated corner radii throughout the interface
- Visual polish and refinements across all screens
- Better integration with iOS 26 design system

## New Features

### Running Timer Indicator
- New UI element for active timer visibility
- Quick access to currently running timer
- Improved at-a-glance timer status

### Per-Client Invoice Settings
- Remembers key invoice settings for each client
- Stores due date preferences
- Saves "Add Tracked Time" preferences
- Settings sync across all devices

### AI Invoice Summaries
- Available on iOS 26 and above
- Automatically summarizes detailed time entry notes
- Generates concise descriptions for invoices
- Saves time when creating client invoices
