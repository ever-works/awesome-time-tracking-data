## Overview

Stretchly is a free, open-source break reminder application available for Windows, macOS, and Linux. It's designed to remind you to take breaks from your computer through configurable mini-breaks and longer breaks, with a lightweight, simple interface.

## Features

### Configurable Breaks
- Mini-breaks (short, frequent breaks for eyes and hands)
- Long breaks (longer rest periods)
- Fully customizable break intervals
- Adjustable break duration
- Multiple break schedules

### Flexibility
- Pause breaks temporarily
- Skip breaks when needed
- Postpone upcoming breaks
- Natural break detection
- Do Not Disturb integration

### Break Activities
- Suggested stretching exercises
- Eye rest reminders
- Microbreak ideas displayed during breaks
- Customizable break messages

### Lightweight Design
- Minimal system resource usage
- Simple, clean interface
- Unobtrusive notifications
- Runs in system tray/menu bar

### Open Source
- Completely free
- Active development community
- Transparent codebase on GitHub
- No telemetry or data collection
- Cross-platform compatibility
