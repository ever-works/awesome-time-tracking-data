## Overview

Serene is a comprehensive focus app for Mac that blocks distractions, provides focus music, and structures work sessions around clear goals to enable deep, productive work.

## Features

- **Website & App Blocking**: Block distracting sites and apps during focus sessions
- **Phone Call Blocking**: Silence phone calls automatically
- **Focus Music**: Curated playlists for concentration
- **Goal-Based Sessions**: Structure work around specific outcomes
- **Session Planning**: Plan deep work blocks
- **Break Reminders**: Scheduled rest periods
- **Daily Reviews**: Track accomplishments

## Workflow

1. Set clear session goal
2. Choose focus duration
3. Select distractions to block
4. Start focus music
5. Begin deep work
6. Take scheduled breaks
7. Review outcomes

## Distraction Management

- Block specific websites
- Block applications
- Silence notifications
- Block phone calls
- Customizable block lists
- Emergency override options
