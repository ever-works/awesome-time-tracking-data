## Overview

solidtime is a modern, open-source time tracking application designed for freelancers and agencies. It can be hosted on your own infrastructure, giving complete control over data and privacy. The platform includes comprehensive features for project management, task tracking, and client billing.

## Key Features

- **Self-hosted option** - Host on your own infrastructure for complete data control
- **Open source** - Free and transparent codebase available on GitHub
- **Project management** - Organize work by clients and projects
- **Task tracking** - Create and track tasks within projects
- **Billable rates** - Set different billing rates for clients, projects, or tasks
- **Time tracking** - Track time with timers or manual entry
- **Reporting** - Generate reports for billing and analysis
- **Data import** - Import existing data from Toggl and Clockify
- **Modern interface** - Contemporary, user-friendly design
- **Multi-user support** - Team collaboration and role-based access

## Data Import Capabilities

solidtime can import time tracking data from:
- Toggl Track
- Clockify

This makes migration from other platforms straightforward.
