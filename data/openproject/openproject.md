## Overview

OpenProject is open-source project management software that supports classic, agile, and hybrid project management approaches. It provides comprehensive features including task management, Gantt charts, boards, team collaboration, and time and cost reporting—all available for self-hosting or as a cloud service.

## Key Features

### Project Management Approaches

**Classic Project Management**
- Gantt charts for timeline visualization
- Project phases and milestones
- Dependencies and critical path
- Resource planning

**Agile Project Management**
- Scrum and Kanban boards
- Sprint planning and backlogs
- Burndown charts
- Velocity tracking

**Hybrid Approaches**
- Combine classic and agile methods
- Flexible workflow adaptation

### Time and Cost Reporting

- **Time tracking** - Log time against tasks and projects
- **Cost tracking** - Monitor project costs and budgets
- **Budget management** - Set and track project budgets
- **Reporting** - Comprehensive time and cost reports
- **Invoicing support** - Generate reports for client billing

### Collaboration Features

- Team communication and comments
- File attachments and document management
- @mentions and notifications
- Activity streams
- Wiki for documentation
- Meeting agendas and minutes

### Task Management

- Create and assign work packages (tasks)
- Custom fields and workflows
- Priority and status tracking
- Task dependencies
- Subtasks and hierarchies

## Deployment Options

### Self-Hosted
- Install on your own infrastructure
- Complete data ownership and control
- No recurring SaaS fees
- Customization possibilities
- Compliance with data sovereignty requirements

### Cloud
- Managed hosting by OpenProject
- No infrastructure management
- Regular updates and maintenance
- Quick setup and deployment

## Open Source Benefits

- **Free to use** - No licensing costs for self-hosted
- **Transparent** - Full source code access
- **Customizable** - Modify to fit specific needs
- **Community-driven** - Active development community
- **No vendor lock-in** - Own your data and instance
- **Plugin ecosystem** - Extend functionality
