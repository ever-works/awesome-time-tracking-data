## Overview

Project Time Budget Alerts provide early warning when projects risk exceeding allocated hours, enabling intervention before budget depletion and profit erosion.

## Alert Thresholds

### Common Warning Levels

**75% Threshold (Yellow)**:
- Heads-up notification
- Review project pace
- Assess remaining work
- Plan adjustment if needed

**90% Threshold (Orange)**:
- Urgent attention required
- Immediate review meeting
- Scope/resource decisions
- Client communication prep

**100% Threshold (Red)**:
- Budget exhausted
- Stop work or approve overage
- Formal change order
- Senior management involvement

**110%+ Threshold (Critical)**:
- Project over budget
- Profitability impacted
- Corrective action required
- Post-mortem planning

## Alert Recipients

**Project Manager**: All thresholds
**Team Members**: At 90% (heads-up)
**Account Manager**: At 75% (client prep)
**Finance/Leadership**: At 100% (budget impact)
**Timesheet Users**: Real-time when entering time
