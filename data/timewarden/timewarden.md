## Overview

TimeWarden is an open-source, self-hosted time tracking application designed for teams and individuals who want full control over their data. It provides comprehensive time tracking with project management features.

## Features

- **Time Tracking**: Manual and timer-based tracking
- **Project Management**: Organize time by projects and tasks
- **Team Tracking**: Multi-user support
- **Reporting**: Detailed time reports
- **Self-Hosted**: Full data ownership and privacy
- **API**: REST API for integrations
- **Export**: Export time data in multiple formats
- **Permissions**: Role-based access control
- **Customizable**: Open source allows modifications

## Deployment

- Docker support
- Self-hosted on your own infrastructure
- No external dependencies on cloud services
- Full data privacy and control
