## Overview

Nirvana is a sophisticated GTD (Getting Things Done) application built specifically for experienced practitioners of David Allen's methodology. It offers comprehensive features that support advanced GTD workflows, including Areas of Focus, multiple review horizons, and detailed context management.

## Features

### Advanced GTD Implementation
- **Six Horizons of Focus**: Support for all GTD levels from ground-level actions to life purpose
- **Areas of Responsibility**: Organize projects by life/work areas
- **Projects**: Detailed project planning with outcomes and next actions
- **Next Actions**: Context-filtered, immediately actionable tasks
- **Waiting For**: Track delegated items with contact information
- **Someday/Maybe**: Incubate future possibilities

### Context System
- **Multiple Contexts**: Assign tasks to locations, tools, people, or energy levels
- **Context Filtering**: View only relevant actions for current situation
- **Time Availability**: Filter by available time (@15min, @1hour)
- **Energy Levels**: Match tasks to current energy (@high-energy, @low-energy)
- **Custom Contexts**: Create unlimited personal contexts

### Project Management
- Parallel and sequential project types
- Project outcomes and purposes
- Next action identification
- Project support materials
- Project templates for recurring workflows
- Multi-level project organization

### Review System
- **Daily Review**: Quick check of calendar and next actions
- **Weekly Review**: Comprehensive system maintenance
- **Monthly Review**: Higher perspective planning
- **Quarterly Review**: Goals and areas review
- **Annual Review**: Life and career planning
- **Review Checklists**: Guided review processes

### Capture & Processing
- Quick capture from any screen
- Email to Nirvana for inbox items
- Processing workflow with clarifying questions
- Batch processing tools
- Reference material storage

### Organization Features
- Tags for flexible categorization
- Scheduled actions and recurring tasks
- Due dates with warnings
- Priority levels
- Notes and attachments
- Contact management for waiting items

## Platform Support

- Web application
- Desktop applications (Windows, macOS, Linux)
- Mobile apps (iOS, Android)
- Cross-device synchronization
- Offline access

## Advanced Features

- **Focus Lists**: Curated lists for specific work sessions
- **Perspectives**: Save custom views and filters
- **Smart Lists**: Dynamic lists based on criteria
- **Keyboard Shortcuts**: Efficient navigation and task creation
- **Quick Add**: Rapid task entry with smart parsing

## For Experienced GTD Users

Nirvana is recommended for practitioners who:
- Have mastered basic GTD principles
- Need sophisticated area and horizon management
- Use multiple contexts extensively
- Maintain detailed project portfolios
- Perform regular multi-level reviews
- Require advanced filtering and perspectives

## Use Cases

- **GTD Veterans**: Long-time practitioners of the methodology
- **Complex Roles**: Professionals managing diverse responsibilities
- **Multi-Project Leaders**: Those overseeing numerous initiatives
- **Consultants**: Managing client work across contexts
- **Executives**: High-level perspective management

## Philosophy

Nirvana believes in providing depth over simplicity for users who understand GTD thoroughly. The application doesn't compromise features to achieve minimalism, instead offering comprehensive tools for serious practitioners.

## Learning Curve

Nirvana has a steeper learning curve than simpler GTD apps but rewards investment with powerful workflow capabilities for dedicated users.
