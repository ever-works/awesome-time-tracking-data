## Overview

LogWork is a simple time tracker that offers automatic time tracking, mainly for desktop. It tracks time to all your projects, clients, and team tasks. The desktop app tracks time in the background, and all tracked data is available in the dashboard for real-time checking.

## Features

- Automatic time tracking via desktop app (background tracking)
- Manual time entry editing
- Real-time dashboard with activity data
- Reports with tracked activities and time logs
- Export reports including Excel export to timesheets
- Client and team member naming
- Idle time detection
- Browser extension with 74+ integrations
- Zapier integration (3000+ apps)
- API for custom integrations
