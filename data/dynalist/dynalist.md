## Overview

Dynalist is described as the most direct functional upgrade from WorkFlowy—maintaining the same infinite outliner concept while adding rich text formatting, dates, file attachments, Google Drive integration, desktop apps, and better mobile experiences.

## Key Features

- **Infinite Outlining**: Unlimited nested levels for hierarchical organization
- **Rich Text Formatting**: Bold, italic, code blocks, and inline formatting
- **Date & Time Support**: Add deadlines and scheduled dates to items
- **File Attachments**: Embed files directly in outline items
- **Google Drive Integration**: Backup and sync with Google Drive
- **Desktop Applications**: Native apps for Windows, Mac, and Linux
- **Mobile Apps**: Improved mobile experience compared to WorkFlowy
- **Tags**: Organize items across different documents
- **Search**: Powerful search across all documents
- **Bookmarks**: Quick access to frequently used documents

## Advanced Capabilities

- **Document Linking**: Reference other documents within outlines
- **Checkboxes**: Convert items to tasks with completion tracking
- **Color Labels**: Visual organization with colored tags
- **Latex Support**: Mathematical equations and formulas
- **Code Blocks**: Syntax highlighting for code snippets
- **Zoom**: Focus on specific branches of your outline
- **Hoisting**: Temporarily treat any item as the root
- **Filters**: View only tagged or dated items

## Organization System

- **Documents**: Separate outlines for different projects/topics
- **Folders**: Organize documents into hierarchical folder structure
- **Favorites**: Star important documents for quick access
- **Inbox**: Default capture location for quick entry
- **Search Filters**: Save complex searches for recurring use

## Collaboration Features

- **Sharing**: Share documents with view or edit permissions
- **Real-Time Collaboration**: Multiple users editing simultaneously
- **Comments**: Add discussion threads to specific items
- **Version History**: Track changes over time (Pro)

## Use Cases

- **Note-Taking**: Capture and organize information hierarchically
- **Project Planning**: Break down projects into nested tasks
- **Writing**: Draft articles, books, or scripts with collapsible sections
- **Research**: Organize research materials with references
- **Meeting Notes**: Capture discussions with action items
- **Knowledge Management**: Build personal wiki or second brain
- **Brainstorming**: Quickly capture and reorganize ideas

## Keyboard Shortcuts

Dynalist maintains WorkFlowy's keyboard-centric approach:
- Tab/Shift-Tab: Indent/outdent
- Ctrl/Cmd+Up/Down: Move items
- Ctrl/Cmd+Enter: Zoom into item
- Ctrl/Cmd+K: Insert link
- And many more for power users
