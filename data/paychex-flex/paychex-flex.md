## Overview

Paychex Flex is Paychex's flagship offering that provides standard pay processing tools SMBs need, including access to a single platform with multiple options to input and view payroll and time tracking data. It represents a comprehensive solution for small and medium-sized businesses seeking integrated HR and payroll capabilities.

## Time Tracking Features

- **Multiple Input Methods**: Various ways to clock in/out and submit time
- **Integrated Timesheets**: Directly connected to payroll processing
- **Mobile Time Tracking**: Track time from mobile devices
- **Automated Calculations**: Eliminate manual time calculations
- **Approval Workflows**: Manager review before payroll processing

## AI-Powered Innovation

### Voice Assist Technology
Paychex takes AI tools further with its Voice Assist solution, allowing employees and managers to interact with the system using natural language voice commands for time-related queries and actions.
