## Overview

Tomato Timer is a straightforward, no-frills Pomodoro timer website. It provides the classic 25-minute work timer with 5-minute breaks, accessible from any browser without installation or signup.

## Features

- **Classic Pomodoro**: 25/5/15 minute intervals
- **One-Click Start**: No configuration needed
- **Browser-Based**: Works on any device
- **No Signup**: Start immediately
- **Customizable**: Adjust timer lengths
- **Notifications**: Audio alerts for breaks
- **Fullscreen Mode**: Minimize distractions
- **Keyboard Shortcuts**: Space to start/pause

## How to Use

1. Visit tomato-timer.com
2. Click start
3. Work for 25 minutes
4. Take 5-minute break
5. Repeat
6. After 4 pomodoros, take 15-minute break
