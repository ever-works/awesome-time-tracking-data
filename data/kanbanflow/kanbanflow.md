## Overview

KanbanFlow combines visual Kanban task management with an integrated Pomodoro timer, allowing you to organize work visually while tracking time spent on tasks using focus sessions.

## Features

- **Kanban Boards**: Visual task organization
- **Integrated Pomodoro**: Built-in timer for tasks
- **Time Tracking**: Track time per task and board
- **Swimlanes**: Organize tasks by category
- **WIP Limits**: Limit work in progress
- **Recurring Tasks**: Automated task repetition
- **Collaboration**: Team boards and assignments
- **Reports**: Time and productivity analytics
- **Mobile Apps**: iOS and Android
- **Integrations**: Zapier and others
