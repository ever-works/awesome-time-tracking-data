# Task Force

Task Force is a paid service offered by Epiforge Software that provides synchronization capabilities for the Grindstone time tracking application.

## Features

- Synchronizes time tracking data across multiple users
- Integrates with Grindstone's task list and timer
- Enables collaborative time tracking for teams
- Hosted service managed by Epiforge Software

## Pricing

Task Force is a paid service; specific pricing details are available on the Epiforge website.

## Requirements

- Requires Grindstone installed on each client machine
- Compatible with Microsoft Windows operating system

## Availability

As of the latest information, Task Force is available for purchase through Epiforge Software's official channels.
