## Overview

Pomotimer is a simple Pomodoro timer designed specifically for focus. It provides a clean, straightforward approach to time management without unnecessary features or distractions.

## Core Philosophy

- Simplicity first
- Focus-oriented design
- No feature bloat

## Key Features

- Classic Pomodoro intervals (25/5)
- Simple timer controls
- Clean visual design
- No account required
- Free to use
- Browser-based

## How It Works

1. Visit pomotimer.com
2. Start 25-minute focus session
3. Work without interruption
4. Take 5-minute break
5. Repeat cycle
6. Long break after 4 Pomodoros
