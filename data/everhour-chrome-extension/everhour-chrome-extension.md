## Overview

Everhour's Chrome extension embeds time tracking directly into your favorite project management and collaboration tools, allowing you to track time without leaving your workflow.

## Native Integrations

The extension adds time tracking buttons directly within:
- Asana
- Trello
- Jira
- Basecamp
- GitHub
- GitLab
- ClickUp
- Monday.com
- Notion
- Todoist
- And 30+ other tools

## Features

- One-click timer start/stop
- Inline time tracking in task cards
- Manual time entry
- Task time estimates
- Project budgets tracking
- Time reports and analytics
- Team time tracking
- Billable hours management
- Invoice generation
- Calendar view
- Dashboard overview

## Time Tracking Capabilities

- Track time on specific tasks
- Add time to multiple tasks
- Edit tracked time
- Add notes and descriptions
- Tag time entries
- Set hourly rates
- Mark billable/non-billable
- Lock time entries after approval

## Reporting

- Detailed time reports
- Budget vs. actual tracking
- Team productivity insights
- Client and project reports
- Export to Excel, CSV
- Custom report filters
- Visual charts and graphs

## Team Features

- Timesheet approvals

- Team capacity planning
- Project cost tracking
