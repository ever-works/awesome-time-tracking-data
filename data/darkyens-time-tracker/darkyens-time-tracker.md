## Overview

Darkyen's Time Tracker is a lightweight, open-source time tracking plugin for JetBrains IDEs. It provides unobtrusive time tracking through a simple status bar widget, making it ideal for developers who want to track project time without disrupting their workflow.

## Features

- **Status Bar Widget**: Displays current project time in the IDE status bar
- **Automatic Tracking**: Tracks time spent on projects based on IDE activity
- **Lightweight Design**: Minimal resource usage and interface footprint
- **Project-Based Tracking**: Separate time tracking for each project
- **JetBrains Integration**: Native support for IntelliJ IDEA, PyCharm, WebStorm, and other JetBrains IDEs
- **Open Source**: Free and open-source software with community support

## Use Cases

- Developers wanting simple, unobtrusive time tracking
- Freelancers tracking time across multiple client projects
- Teams standardizing on lightweight time tracking tools
- Individual developers analyzing time allocation patterns
