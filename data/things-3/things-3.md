## Overview

Things 3 is a beautifully designed task management app for Apple devices that combines powerful GTD-inspired features with an exceptionally intuitive interface. It's known for its elegant design, smooth user experience, and perfect balance between simplicity and capability.

## Features

- **Today View**: Focus on what's important right now
- **Upcoming**: Calendar-integrated view of scheduled tasks
- **Anytime**: Tasks to do when you have time
- **Someday**: Ideas and tasks for the future
- **Projects**: Group related tasks with optional headings
- **Areas**: Organize ongoing responsibilities
- **Tags**: Flexible categorization and filtering
- **Checklists**: Subtasks within tasks
- **Deadlines**: Separate deadlines from scheduled dates
- **Quick Entry**: System-wide shortcut for rapid capture
- **Magic Plus**: Natural language task creation
- **Repeating Tasks**: Flexible recurring task patterns
- **Sync**: Seamless sync via Things Cloud
- **Widgets**: Home screen and Today widgets
- **Siri Integration**: Voice task creation
- **Handoff**: Continue between devices seamlessly

## Design Philosophy

- **Minimal**: Clean interface with no clutter
- **Fast**: Instant response, optimized performance
- **Accessible**: Easy to learn, no steep learning curve
- **Beautiful**: Winner of Apple Design Awards
- **Private**: All data stored on your devices, not sold

## GTD-Inspired Workflow

- **Inbox**: Capture everything quickly
- **Organize**: File into projects and areas
- **Schedule**: Assign dates when tasks become relevant
- **Review**: Evening Review and This Evening features
- **Execute**: Focus on Today list

## Limitations

- Apple ecosystem only (no Windows, Android, web)
- No collaboration or sharing features
- No built-in time tracking
- One-time purchase (no subscription, but also no cross-platform)

## Pricing

- **Mac**: $49.99 (one-time purchase)
