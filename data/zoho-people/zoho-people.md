## Overview

Zoho People is a cloud-based HR management system specializing in employee monitoring and time tracking software to enhance organizational productivity. The platform is designed to minimize burnout and boost engagement across remote, hybrid, and in-office work environments.

## Key Features

**Attendance Tracking:**
- Real-time attendance capture
- Web and mobile app check-ins
- Biometric integrations
- People Kiosk with facial recognition
- IP and geo-fencing for mobile workforce
- Multiple tracking input methods

**Time Management:**
- Integration with time tracker
- Log daily, weekly, and monthly work hours
- Billable and non-billable hour tracking
- Accurate payroll calculations
- Shift rules configuration
- Overtime monitoring
- Real-time attendance dashboards

**Attendance Corrections:**
- Regularization for manual errors
- Missed entries resolution
- Approval workflows for corrections
- Easy error resolution

**Geo-Fencing & Location:**
- Check-ins only in approved locations
- Mobile workforce agility
- IP restriction options
- Location verification

**Shift Management:**
- Set up shift rules
- Track overtime automatically
- Monitor attendance patterns
- Flexible shift configurations
