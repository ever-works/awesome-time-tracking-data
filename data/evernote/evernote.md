## Overview

Evernote is a long-standing note-taking and information management platform that enables users to capture, organize, and retrieve information across all devices. It supports rich text, images, audio recordings, web clips, and scanned documents, serving as a centralized knowledge repository for individuals and teams.

## Key Features

- **Rich note creation** — Text, images, audio, and file attachments in a single note
- **Web Clipper** — Browser extension to save web pages, articles, and screenshots
- **Document scanning** — Mobile camera integration for scanning physical documents
- **Notebooks and tags** — Hierarchical organization with notebooks, stacks, and tags
- **Cross-device sync** — Seamless access across desktop, web, iOS, and Android
- **Search capability** — Full-text search including text within images and PDFs
- **Collaboration** — Shared notebooks and note sharing for team use
- **Templates** — Pre-built templates for meeting notes, project planning, and more
- **AI capabilities** — AI-powered summary and search assistance

## Pricing

Free plan with limited devices and basic features. Premium and Professional plans available with increased storage, offline access, and advanced features.
