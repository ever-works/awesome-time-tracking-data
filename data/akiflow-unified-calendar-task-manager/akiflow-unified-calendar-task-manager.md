## Overview

Akiflow is a unified calendar and task manager that consolidates all your calendars and tasks into one interface, enabling effective time blocking and daily planning.

## Key Features

### Calendar Consolidation
View multiple calendars in one place:
- Google Calendar
- Outlook Calendar
- Apple Calendar
- All events synced in real-time

### Task Integration
Pull tasks from multiple sources:
- Todoist
- Notion
- Asana
- ClickUp
- Jira
- Slack
- Email

### Time Blocking
- Drag tasks onto calendar to schedule
- Visual time allocation
- See tasks and meetings together
- Plan entire day in one view

### Command Bar
Quick keyboard-driven task and event creation for power users.

## Platform Support
