## Overview

Flat Tomato is crafted to fulfill every need of Pomodoro users with a focus on beautiful design and user experience.

## Features

- **Beautiful Design**: Modern, flat design interface
- **Task Management**: Set and organize tasks with timers
- **Multiple Intervals**: Customizable work and break periods
- **Ambient Music**: Built-in background music and sounds
- **Focus Sounds**: Various ambient noises to enhance concentration
- **Statistics**: Track completed Pomodoros and productivity trends
- **Notifications**: Alerts for session changes
- **Dark Mode**: Eye-friendly interface for any time of day
