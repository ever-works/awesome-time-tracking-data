## Overview

Default Diary creates ideal weekly template showing desired time allocation across all priorities.

## Creation Process

1. List all life categories
2. Assign ideal hours to each
3. Block template calendar
4. Include work, personal, relationships, health
5. Use as starting point each week

## Benefits

- Makes priorities explicit
- Reveals actual vs intended gaps
- Provides structure for planning
- Shows realistic capacity
- Enables intentional adjustments

## Usage

- Copy template each week
- Adjust for specific commitments
- Compare actual vs template weekly
