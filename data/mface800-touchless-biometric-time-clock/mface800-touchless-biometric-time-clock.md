## Overview

The mFace800 is a sophisticated touchless biometric time clock that combines facial recognition with fingerprint authentication. It offers fast, accurate, and hygienic employee time tracking with advanced anti-spoofing technology.

## Features

- Touchless facial recognition (primary)
- Fingerprint biometric backup
- Fast recognition speed (less than 0.5 seconds)
- Anti-spoofing technology prevents photo/video fraud
- Liveness detection
- Large capacity (10,000 faces, 10,000 fingerprints)
- 8-inch touchscreen display
- WiFi and Ethernet connectivity
- Access control integration
- Real-time data sync
- Employee self-service functions
- Integration with time and attendance software
- Optional temperature screening
- Audit trail logging
