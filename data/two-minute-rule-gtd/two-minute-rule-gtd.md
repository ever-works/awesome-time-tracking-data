## Overview

The 2-Minute Rule is a decision-making principle from David Allen's Getting Things Done (GTD) methodology that provides a simple guideline for handling quick tasks.

## The Rule

When processing your inbox (physical, email, or mental), if an action will take less than 2 minutes, do it immediately instead of deferring it.

## Why It Works

- **Context Switching Cost**: It takes more time to capture, organize, and later retrieve a 2-minute task than to just do it now
- **Decision Fatigue**: Eliminates recurring decisions about the same small task
- **Momentum**: Completing quick tasks creates positive momentum
- **Clean Lists**: Keeps to-do lists focused on meaningful work

## When to Apply

### Do It Now (Under 2 Minutes)
- Reply to simple yes/no emails
- File a document
- Make a quick phone call
- Update a spreadsheet cell
- Send a calendar invite
- Write a brief Slack message
- Schedule an appointment

### Defer It (Over 2 Minutes)
- Detailed email responses
- Research tasks
- Writing documents
- Complex problem-solving
- Anything requiring focused thought

## Common Mistakes

### Misjudging Time
**Problem**: "2 minutes" becomes 10 minutes
**Solution**: Practice time estimation; be ruthless about the 2-minute cutoff

### Constant Interruption
**Problem**: Doing 2-minute tasks all day prevents deep work
**Solution**: Batch process inputs 2-3 times daily; protect deep work time

### Priority Blindness
**Problem**: 2-minute tasks interrupt high-priority work
**Solution**: Apply rule only during processing time, not during focused work

## Advanced Application

### Email Processing
When reviewing email, immediately:
- Reply to simple requests
- Forward to appropriate person
- File for reference
- Delete if not needed
- Only defer if response requires >2 minutes

### Meeting Follow-Up
After meetings, spend 2 minutes:
- Sending thank-you notes
- Sharing quick updates
- Scheduling follow-up meetings

## Modifications

### The 5-Minute Version
Some productivity experts extend to 5 minutes for tasks with high impact.

### The 30-Second Rule
For extremely busy periods, only do tasks under 30 seconds immediately.

## Benefits

- Cleaner, more focused task lists
- Reduced mental clutter
- Fewer small tasks accumulating
- Better responsiveness to simple requests
- More time for meaningful work
