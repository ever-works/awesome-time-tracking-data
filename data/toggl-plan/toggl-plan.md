## Overview

Toggl Plan (formerly Teamweek) provides visual project planning with drag-and-drop timelines. It helps teams plan projects, allocate resources, and balance workloads alongside Toggl Track's time tracking.

## Features

- Visual timeline planning
- Drag-and-drop task scheduling
- Team capacity planning
- Milestone tracking
- Color-coded projects
- Team availability view
- Public sharing links
- Integration with Toggl Track

## Use Cases

- Project scheduling
- Resource allocation
- Team workload balancing
- Sprint planning
- Campaign planning

## Integration with Toggl Track

- Plan projects in Toggl Plan
- Track actual time in Toggl Track
- Compare planned vs. actual
- Unified reporting
