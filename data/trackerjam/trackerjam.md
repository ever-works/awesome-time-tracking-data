## Overview

TrackerJam is an automatic browser web time tracking solution that analyzes team browser activity to provide detailed insights into core time metrics and productivity reports.

## Components

TrackerJam is a combination of:
- **Chrome Extension**: Analyzes browser activity and tracks time spent on websites and applications
- **Web Dashboard**: Displays detailed reports and analytics

## Features

- Automated time tracking through browser extension
- Team browser activity analysis
- Detailed productivity insights and reports
- Core time metrics tracking
- Website and application usage analytics
- Team collaboration features

## Use Cases

Ideal for teams and organizations that want to:
- Monitor browser-based productivity
- Understand how time is allocated across different websites and applications
- Analyze team work patterns
- Generate time-based reports for clients or projects
- Track web-based task completion

## Platform

Available as a Chrome extension with accompanying web dashboard for report viewing and team management.
