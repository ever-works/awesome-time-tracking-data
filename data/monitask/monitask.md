## Overview

Monitask is employee monitoring software that allows employees, field contractors, and freelancers to manually clock in when they begin working. The application emphasizes transparency by only tracking activity when employees are clocked in, and can only be installed with the user's permission.

## Features

### Time Tracking
- Manual clock-in/clock-out functionality
- Automatically calculated hours
- Eliminates need for manual timesheets
- Simple click to start and end shifts

### Screenshot Monitoring
- Proof-of-work screenshots taken randomly or at set intervals
- Review daily operations
- Identify unproductive employees
- Screenshots can be turned off or blurred for sensitive content

### Activity Monitoring
- Track programs employees use during work time
- Reports for each program used and duration
- Internet monitoring (Business edition)
- Application monitoring (Business edition)

### Task Management
- Assign and track tasks
- Task completion monitoring
- Team analytics

### Platform Availability
- Windows
- macOS
- Linux
- Cross-platform support

## Privacy Approach

- Only tracks when employee is clocked in
- Promotes transparency over spying
- Requires user permission for installation
- Not spy software - employees control when tracking starts

## Integrations

- Asana integration available
- Unlimited integrations (Business edition)
