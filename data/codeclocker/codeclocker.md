## Overview

CodeClocker is a specialized time tracking plugin for JetBrains IDEs designed to help developers and teams track coding time with precision and generate accurate invoices.

## Features

- **JetBrains Integration**: Native plugin support for IntelliJ IDEA, PyCharm, WebStorm, and other JetBrains IDEs
- **Automatic Time Tracking**: Tracks coding sessions automatically based on IDE activity
- **CSV Export**: Export coding activity to CSV format for easy invoicing and time reporting
- **Detailed Reports**: Reports include project names, branches, hours worked, and commit references
- **Branch-Level Tracking**: See time spent on specific branches for detailed project insights
- **Commit References**: Link time tracking to specific commits for accurate project documentation

## Use Cases

- Freelance developers generating invoices based on coding time
- Development consultants providing detailed time reports to clients
- Software teams tracking time allocation across different projects and features
- Individual developers analyzing productivity patterns
