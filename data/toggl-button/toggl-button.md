## Overview

Toggl Button (formerly Toggl Browser Extension) is a free browser extension that adds time tracking functionality directly into the web apps you already use, making time tracking seamless and effortless.

## Features

- **100+ Integrations**: Works with Asana, Trello, Jira, GitHub, Gmail, etc.
- **One-Click Tracking**: Start timer without leaving current app
- **Auto-Fill**: Automatically captures task names and projects
- **Pomodoro Timer**: Built-in Pomodoro option
- **Idle Detection**: Detects when you're away
- **Reminders**: Prompts to track time
- **Syncs with Toggl Track**: All data syncs to main Toggl app
- **Cross-Browser**: Chrome, Firefox, Edge
- **Keyboard Shortcuts**: Quick timer control

## How It Works

1. Install browser extension
2. Log in to Toggl account
3. Visit supported web app (e.g., Asana)
4. Click Toggl button next to tasks
5. Time tracking starts automatically
6. Data syncs to Toggl Track
