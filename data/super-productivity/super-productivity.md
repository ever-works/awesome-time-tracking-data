## Overview

Super Productivity is an open-source to-do list and time tracker that integrates task management with time tracking, Pomodoro timers, and timeboxing features for enhanced productivity.

## Features

- **Task Management** - Create and organize tasks with project categorization
- **Time Tracking** - Track time spent on tasks with built-in timers
- **Pomodoro Timer** - Integrated Pomodoro technique support
- **Timeboxing** - Implement time blocking and timeboxing methods
- **Issue Tracker Integration** - Connect with Jira, GitHub, GitLab, and more
- **Offline First** - Works completely offline with local data storage
- **Cross-Platform** - Available for Windows, Mac, Linux, Android, and web
- **Open Source** - Free and open-source with active community development
- **Privacy** - All data stored locally with optional sync

## Use Cases

Ideal for developers, freelancers, and productivity enthusiasts who want an open-source solution combining task management with time tracking.
