## Overview

Tim is a simple, straightforward time tracking application designed exclusively for macOS. It embodies minimalist design principles, offering time tracking functionality without unnecessary features while maintaining user privacy by storing all data locally on the device.

## Key Features

- **Menu Bar Integration**: Time displayed directly in the macOS menu bar for constant visibility
- **Keyboard Shortcuts**: Control time tracking entirely via keyboard shortcuts for efficiency
- **Local Data Storage**: All data remains on your device—no accounts or internet connection required
- **Task Organization**: Create, track, and edit tasks and time records with support for groups and tags
- **Visual Analytics**: Beautiful charts providing visual breakdown of hours, averages, and revenue
- **Flexible Filtering**: Filter time by date ranges and/or tags for detailed analysis
- **Export Options**: Export data to CSV, JSON, or iCal formats
- **Offline Operation**: Works completely offline with no cloud dependency

## Privacy & Security

Tim prioritizes user privacy:
- No user accounts required
- No data sent to external servers
- Complete offline functionality
- Full user control over data

## Use Cases

- Freelancers tracking billable client hours
- Developers monitoring time spent on different projects
- Anyone wanting simple time tracking without cloud services
- Users who value privacy and local data storage
- macOS users preferring native menu bar applications

## Pricing

Tim is available as a free download with limited functionality. The Tim Pro upgrade is available as a one-time in-app purchase of approximately $20, with no ongoing subscription costs.

## Platform

Available exclusively for macOS.
