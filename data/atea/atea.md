## Overview

Aeta is a minimalistic text file based menu bar time tracker for MacOS. The current task file resides in a configurable location, and tracked times are stored in CSV files.

## Key Features

- **Text File Based**: All data stored in simple, portable CSV format
- **Menu Bar Access**: Quick access from the macOS menu bar
- **Local Storage**: Current task file and time logs reside in configurable local locations
- **No Database**: Avoid complexity of database systems with plain text storage
- **Minimal Footprint**: Lightweight application with minimal system resource usage

## Philosophy

Designed for users who prefer simplicity and data portability over feature richness, Atea provides just enough functionality for basic time tracking without unnecessary complexity.

## Data Format

Tracked times are stored in CSV (Comma-Separated Values) format, making it easy to import into spreadsheets, databases, or custom analysis tools.

## Platform

Available for macOS.
