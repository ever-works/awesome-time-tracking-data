# DueFocus

DueFocus is a time tracking and productivity tool tailored for developers and software teams, with task-focused tracking and coding analytics, available as a Chrome app via the Chrome Web Store.

## Features
- **Developer-focused time tracking** – Designed specifically for developers and software teams.
- **Task-based tracking** – Log time against specific tasks to understand how effort is distributed.
- **Coding analytics** – Provides analytics around coding activity to help assess productivity.
- **GitHub-linked time entries** – Link time entries directly to GitHub issues and commits.
- **Work insights for planning** – Helps teams understand where development time is spent to improve project estimation and planning.
- **Chrome app** – Distributed as a Chrome app, installable via the Chrome Web Store.

## Integrations
- **GitHub** – Associate tracked time with issues and commits for more detailed development reporting.
