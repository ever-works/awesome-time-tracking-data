## Overview

TeamOB provides architecture project management and time tracking software that seamlessly integrates with Revit, AutoCAD, SketchUp, and other BIM tools to automatically track design time and project phases.

## CAD/BIM Integration

### Automatic Time Tracking

- Integrates directly with Revit for real-time tracking
- Connects with AutoCAD for design time capture
- Works with SketchUp for 3D modeling time
- Supports other BIM and CAD tools
- Automatically categorizes time by software and project

### Design Phase Monitoring

Track time spent on:

- Conceptual design
- Schematic design
- Design development
- Construction documents
- Technical documentation
- Revisions and iterations

## Detailed Activity Tracking

- Captures time spent on drawings
- Monitors revisions and changes
- Tracks technical documentation time
- Records meeting and coordination time
- Provides detailed activity breakdowns

## Project Management Features

- Project phase organization
- Task and milestone tracking
- Resource allocation
- Budget monitoring
- Deadline management
- Team collaboration tools
