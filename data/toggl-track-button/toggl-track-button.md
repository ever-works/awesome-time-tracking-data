# Toggl Track Button

**Category:** Time Tracking APIs & Integrations  
**Type:** Open-source browser extension  
**Brand:** Toggl Track

## Overview
Toggl Track Button (now the Toggl Track browser extension) is an open-source browser extension for Chrome and Firefox that embeds a Toggl Track time tracking button into various web tools and services. It lets you start and stop timers directly from the tools you already use, or from the extension’s own menu.

## Features
- **Browser support**
  - Dedicated extension for **Google Chrome** (via Chrome Web Store)
  - Dedicated extension for **Mozilla Firefox** (via Firefox Add-ons)

- **Embedded time tracking buttons**
  - Adds a Toggl Track button directly into supported web apps and services
  - Allows starting and stopping timers from within your existing workflow

- **Extension icon menu**
  - Option to start a time entry directly from the browser extension icon/menu (without going into a specific web app)

- **Wide range of integrations**
  - Integrates with many third-party web tools and services (full list maintained in the repo’s `INTEGRATIONS.md`)

- **Custom domain support**
  - Ability to define **custom domains** for supported services (e.g., self-hosted or company-specific domains such as `https://yourcompany.github.com`)
  - Configuration via **Settings → Integrations** in the extension

- **Open-source and contribution-friendly**
  - Open-source codebase hosted on GitHub
  - Extension architecture updated to make it easier to contribute new integrations
