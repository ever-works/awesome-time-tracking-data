## Overview

Baralga is a simple, lightweight time tracking application designed for individuals and small teams. It focuses on ease of use and fast data entry without unnecessary complexity.

## Features

- Simple time tracking with start/stop timer
- Manual time entry with shortcuts
- Project management
- Activity descriptions
- Filtering and reporting
- Data export (CSV, Excel)
- Multi-platform support (Windows, macOS, Linux)
- Minimal system requirements
- No cloud dependency - works entirely offline

## Philosophy

Baralga follows a minimalist approach to time tracking, providing just the essential features needed to track time effectively without overwhelming users with complex functionality.

## Technical Details

- Built with Java for cross-platform compatibility
- Lightweight desktop application
- Local data storage
- Open-source under GPL license
