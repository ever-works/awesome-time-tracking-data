## Coda Native Automations

Automations in Coda allow you to streamline workflows and automate repetitive tasks, helping with project management, data tracking, and team collaboration to save time and increase productivity. Automations have two required steps: When and Then.

**Trigger Types:**
- Row changed: The automation will start when a row in a specified table changes
- Time-based: The automation will run on a specified cadence (hourly, daily, weekly, or monthly)
- Form submitted: The automation will run when someone submits a form
- Webhook invoked: The automation will run when a POST request is made to a specified URL

**Important limitation:** Columns that contain formulas cannot trigger automations, and the row change trigger can only work with columns that are manually updated (via direct edits, button-generated edits, or API edits).

## Built-in Time Tracker Template

Coda offers a time tracker doc that allows you to track hours spent on different projects or tasks, and it's yours to adapt and evolve for your needs.

## Custom Formula Example

For those building custom time tracking systems, there's a formula approach that filters time slots by date and start time to calculate duration.

## Third-Party Integrations

Several third-party time tracking tools integrate with Coda:
- **TrackingTime**: Allows you to start tracking time right from Coda with a Chrome/Firefox extension
- **Timely**: Tracks time spent in Coda automatically in the background while you write
- **TMetric**: Track time spent working on a document with one mouse click
- **Toggl**: Easy to create time-tracking systems in Coda with Toggl
- **Zapier**: Quickly connect Coda and Timely Time Tracking with over 8,000 apps
