## Overview

Code Time is an open-source plugin that provides automatic programming metrics and time tracking for IntelliJ and JetBrains IDEs. It offers developers insights into their coding patterns and productivity with both free and premium options.

## Features

- **Automatic Tracking**: Passively monitors coding activity across your editors and IDEs
- **Cross-Editor Support**: Works with IntelliJ, PyCharm, WebStorm, and other JetBrains IDEs
- **Programming Metrics**: Track time by language, project, file, and branch
- **Open Source**: Free and open-source core functionality
- **Weekly Reports**: Receive insights about your coding patterns
- **Code Time by Project**: Available in Pro plan for detailed project-level analytics

## Use Cases

- Individual developers tracking coding time and patterns
- Freelancers monitoring billable development hours
- Teams analyzing productivity metrics
- Developers optimizing their coding schedules

## Pricing

- **Free Plan**: Basic time tracking across editors and IDEs
- **Pro Plan**: $9.95/month for advanced reporting including code time by project
