## Overview

FocuTime is a unique iOS Pomodoro timer that sets itself apart by automatically blocking distracting apps during scheduled focus periods using Apple's Screen Time API. Unlike other focus apps, it works even when closed, making it ideal for recurring focus schedules.

## Unique Features

### Automatic App Blocking
- Uses Screen Time API for true app blocking
- Blocks apps even when FocuTime isn't running
- Works at the iOS system level
- Cannot be easily bypassed during focus time
- Restores access automatically after sessions

### Recurring Focus Schedules
- Set up daily or weekly focus blocks
- Automatic activation at scheduled times
- Perfect for consistent routines (e.g., 9-11 AM deep work)
- No need to manually start sessions
- Background scheduling that persists

### Focus Block Types
- Study hours for students
- Deep work blocks for professionals
- Digital detox periods
- Creative time protection
- Meeting-free focus windows

## How It Works

1. **Schedule Creation**: Set recurring focus times in your calendar
2. **App Selection**: Choose which apps to block during focus
3. **Automatic Activation**: FocuTime blocks apps at scheduled times
4. **Hands-Free Operation**: Works without opening the app
5. **Automatic Restoration**: Apps become available when sessions end
