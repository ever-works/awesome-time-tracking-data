## Overview

Minimalist Time Tracker is a Chrome extension that provides straightforward time tracking functionality without overwhelming features. Perfect for those who want simple, fast time tracking integrated into their browser.

## Features

- **Quick Start/Stop**: One-click timer
- **Task Labels**: Categorize time entries
- **Time Log**: View tracking history
- **Export**: Export data to CSV
- **Browser-Based**: Works in Chrome
- **Lightweight**: Minimal resource usage
- **No Account**: No registration required
- **Open Source**: Code available on GitHub
