## Overview

Browser extension time trackers add time tracking buttons and functionality directly to web browsers, allowing users to start timers, log time, and track work without leaving their current webpage or switching to separate time tracking software.

## Key Features

- One-click timer start/stop from browser toolbar
- Automatic integration with web-based tools
- Keyboard shortcuts for quick access
- Pomodoro timer functionality
- Idle time detection
- Automatic time entry from current page
- Offline tracking with sync when connected

## Supported Browsers

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Brave
- Opera

## Integration Capabilities

Most browser extensions automatically detect and integrate with 50+ web applications including:
- Project management (Asana, Trello, Jira)
- Development tools (GitHub, GitLab)
- Communication (Gmail, Outlook)
- Documents (Google Docs, Notion)

## Benefits

- Minimal workflow disruption
- No context switching needed
- Lightweight and fast
- Always accessible in browser
- Automatic page title capture
- Seamless tool integration
