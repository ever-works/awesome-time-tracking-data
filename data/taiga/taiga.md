## Overview

Taiga is a free and open-source project management platform designed for agile teams that need a flexible tool for managing projects using Scrum, Kanban, or hybrid methodologies with integrated time tracking.

## Features

- **Scrum Support**: Full scrum implementation with user stories, sprints, and backlog management
- **Kanban Boards**: Visualize workflow with customizable kanban boards
- **Time Tracking**: Track time estimates and actual hours on user stories and tasks
- **Burndown Charts**: Monitor sprint progress with burndown and burnup charts
- **Epics**: Organize large features with epics that span multiple sprints
- **Custom Fields**: Add custom attributes to user stories, tasks, and issues
- **Wiki**: Built-in wiki for project documentation
- **Issue Tracking**: Manage bugs and technical debt separately from user stories
- **Video Calls**: Integrated video conferencing for remote teams
- **Self-Hosted**: Deploy on your own infrastructure for complete data control

## Pricing

Free and open-source for self-hosted deployments. Cloud-hosted version available with free tier for personal use and paid plans starting at $5 per user per month for teams.
