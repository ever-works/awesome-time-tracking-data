## Overview

Ayoa (formerly Droptask) is a visual collaboration platform that combines mind mapping with task management, offering a unique approach to project planning and creative thinking. The software emphasizes visual organization and is particularly suited for creative professionals and visual thinkers.

## Key Features

- **Mind Mapping**: Create visual mind maps for brainstorming and planning
- **Radial Task Boards**: Unique circular task organization system
- **AI-Powered Tools**: AI assistance for ideation and task generation
- **Visual Task Management**: Organize tasks in visually engaging formats
- **Real-Time Collaboration**: Work simultaneously with team members
- **Multiple View Options**: Mind maps, task boards, Gantt charts, and calendars
- **Speed Mind Mapping**: Rapid capture of ideas and thoughts
- **Whiteboard Mode**: Free-form canvas for brainstorming sessions

## Unique Selling Points

### Radial Task Boards
Ayoa's signature radial layout presents tasks in a circular format, making it easy to see relationships and priorities at a glance - different from traditional linear task lists.

### Mind-to-Task Workflow
Seamlessly transform mind maps into actionable task boards, bridging the gap between ideation and execution.

### Neurodiversity-Friendly Design
Designed with input from neurodivergent users, making it particularly effective for people with ADHD, dyslexia, and other cognitive differences.
