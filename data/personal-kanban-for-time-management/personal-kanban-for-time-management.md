## Overview

Personal Kanban adapts team kanban practices for individual productivity with visual workflow management.

## Core Elements

### Visualize Work
- Make all tasks visible
- Three columns: To Do, Doing, Done
- Physical or digital board

### Limit WIP
- Maximum 3-5 active tasks
- Prevents overcommitment
- Forces completion before starting new work

## Benefits

- Clear view of commitments
- Prevents overwhelm
- Tracks progress visibly
- Encourages finishing
- Reduces context switching

## Implementation

- Use sticky notes or digital tool
- Set WIP limit based on capacity
- Pull work, don't push
- Review and adjust regularly
