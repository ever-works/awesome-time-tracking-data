## Overview

Instapaper is a read-it-later service that allows users to save articles, blog posts, and other web content for later consumption. It presents saved content in a clean, distraction-free format optimized for reading.

## Features

- Save articles and web content for later reading
- Add notes alongside highlights
- Distraction-free, formatted text view
- Works across devices
- Offline reading support
- Ability to organize saved content into folders

## Pricing

Free plan available with core features. Premium plan from $5.99/month with additional features and functionality.
