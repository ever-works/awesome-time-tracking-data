## Overview

Todo.vu is a productivity suite designed for freelancers that integrates task management, time tracking, and billing functionality into a single browser-based application.

## Features

- Task management with ability to record time directly on tasks
- Time tracking with time budget creation
- Customer and billing information storage
- Task highlighting capabilities
- Calendar synchronization with external calendar applications
- Clean, intuitive interface

## Pricing

- Free plan available
- Business Time: $9 USD per user per month

## Pros

- Clean and intuitive interface
- Right balance of essential time tracking features
- Ability to record time directly on tasks
- Time budget creation for better project management
- Calendar sync from other apps

## Cons

- Interface is more generic compared to other time tracking apps
- Only accessible through web browsers (no desktop or mobile apps)
- Occasionally suffers from bugs
