## Overview

A lightweight, privacy-focused time tracking CLI built in Go that automatically monitors productivity with beautiful terminal dashboard, smart reports, and configurable goals.

## Core Philosophy

All data stays local - no cloud required. Complete privacy and data ownership are the primary focus.

## Key Features

- **Automatic Productivity Monitoring**: Tracks time automatically without manual start/stop
- **Beautiful Terminal Dashboard**: Rich, visual interface directly in the terminal
- **Smart Reports**: Intelligent analysis of time usage patterns
- **Configurable Goals**: Set and track productivity targets
- **Local-First**: All data stored locally on your machine
- **No Cloud Sync**: Zero data transmitted to external servers

## Technical Details

- Built with Go/Golang for performance and portability
- Cross-platform compatibility
- Lightweight resource usage
- Command-line interface for power users

## Use Cases

- Developers tracking coding time
- Privacy-conscious individuals
- Offline work environments
- Users who want complete data control
