## Overview

Ever Gauzy is a productivity tool that helps users monitor time spent on tasks while providing insights into work habits and activity patterns. Being an open-source tool, it is highly customizable, allowing anyone to tailor it to their needs under an open-source license.

## Features

- Time tracking for projects and tasks with productivity insights
- Tracks activity levels and generates detailed reports
- Open-source design means users can modify and enhance the tool to suit specific requirements
- Supports integration with other open-source tools and workflows

## Pricing

Free and open-source. The source code is publicly available, meaning anyone can access, modify, and distribute it under an open-source license.
